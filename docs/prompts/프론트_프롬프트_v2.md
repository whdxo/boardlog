# BoardLog 프론트엔드 AI 프롬프트 v2

> Stitch / v0 / Bolt / Lovable 등 AI 프론트엔드 빌더용
> 페이지 1개 = 프롬프트 1개 원칙. 각 프롬프트에서 모바일 버전 + 데스크톱 버전 2개씩 만듦.
> 반드시 순서대로 진행.

## 프롬프트 1 — 디자인 시스템 + 공통 컴포넌트

```
보드게임 종합 플랫폼 "BoardLog"의 프론트엔드를 만들어줘.
한국 사용자 대상이라 모든 텍스트는 한국어로. "BoardLog" 서비스명만 영문 유지.

기술 스택:
- Next.js 14 (App Router), TypeScript, Tailwind CSS, shadcn/ui, Lucide Icons
- 폰트: Pretendard (한국어), Inter (영문/숫자)

디자인 시스템:
- Primary: #3B82F6, hover #2563EB, light #DBEAFE
- Accent: #FBBF24 (별점 노란색)
- Success: #10B981, Error: #EF4444
- Gray: 900 #111827, 600 #4B5563, 400 #9CA3AF, 200 #E5E7EB, 50 #F9FAFB
- 배경: #F9FAFB (페이지), #FFFFFF (카드)
- border-radius: 카드 12px, 버튼/인풋 8px, 태그 9999px (pill)
- shadow-sm: 0 1px 2px rgba(0,0,0,0.05), shadow-md: 0 4px 6px rgba(0,0,0,0.07)

타이포: display 28px Bold, h1 22px Bold, h2 18px SemiBold, h3 16px SemiBold, body 14px Regular, caption 12px Regular

간격: 4px 단위 (4, 8, 12, 16, 24, 32, 48)

화면 크기별 레이아웃 (모바일과 데스크톱 2개 버전을 각각 만들어줘):
- 모바일: ~767px (1열, 좌우 패딩 16px, 하단 탭바)
- 데스크톱: 1024px~ (max-width 1080px, 중앙 정렬, 양쪽 여백 넉넉, 상단 GNB + 푸터)

다크모드 없음. 라이트 모드만.

공통 컴포넌트:
1. Button — Primary(#3B82F6), Secondary(흰 배경+파란 보더), Ghost(배경 없음), Danger(#EF4444) / sm(32px), md(40px), lg(48px)
2. Input — 높이 40px, radius 8px, 포커스 시 #3B82F6 보더, 에러 시 #EF4444 보더+하단 에러 텍스트
3. GameCard — 썸네일(3:4비율, radius 8px) + 게임명(16px SemiBold) + ★평점(12px) + 가격(14px SemiBold, #3B82F6), hover시 translateY(-2px)+shadow-md
4. Tag/Badge — pill 형태(radius 9999px), variant: primary(#DBEAFE), success(#10B981), accent(#FEF3C7), gray(#F3F4F6)
5. StarRating — 5점 만점, 1점 단위, 채워진 별 #FBBF24, 빈 별 #E5E7EB, 선택 시 scale(1.15→1.0) 바운스
6. Toast — 성공(✓ 초록), 에러(✕ 빨강), 정보(ℹ 파랑), 3초 노출 후 fadeOut, 모바일 하단/데스크톱 우상단
7. Modal — 데스크톱: 중앙 max 480px, radius 12px / 모바일: 바텀시트(하단 슬라이드업, 드래그 핸들)
8. Skeleton — 카드형+리스트형, #F3F4F6 배경, opacity 펄스 애니메이션(1.5초)
9. EmptyState — 중앙 정렬, 아이콘(48px) + 메시지(14px #9CA3AF) + CTA 버튼
```

## 프롬프트 2 — 공통 레이아웃

```
BoardLog의 공통 레이아웃을 만들어줘. 모든 페이지가 이 레이아웃 안에 들어감.
모바일 버전(~767px)과 데스크톱 버전(1024px~) 2개 페이지를 각각 만들어줘.

모바일 (~767px):
- 상단 헤더 56px 고정
  - 메인: 좌측 "BoardLog" 로고(18px Bold, #3B82F6) + 우측 알림🔔, 설정⚙️
  - 상세: 좌측 ← 뒤로가기 + 중앙 페이지 제목 + 우측 ⋮
  - 배경 white, 하단 보더 #E5E7EB
- 콘텐츠: flex-1, 스크롤
- 하단 탭바 56px 고정 + safe area
  - 4탭: 홈(Home), 탐색(Dice5), 내기록(BookOpen), 프로필(User)
  - 활성: filled 아이콘 + #3B82F6, 비활성: outline + #9CA3AF
  - 아이콘 24px, 텍스트 11px, 균등 4분할

데스크톱 (1024px~):
- 상단 GNB 64px 고정
  - 좌측: "BoardLog" 로고 (22px Bold, #3B82F6)
  - 중앙: 홈, 탐색, 내기록 (14px, 간격 32px, 활성 #3B82F6 + 하단 2px 바)
  - 우측 로그인 상태: 🔔알림 + 프로필(32px 아바타+닉네임+▾)
  - 우측 비로그인: [로그인] Ghost + [회원가입] Primary sm
  - 배경 white, shadow-sm
- 프로필 드롭다운: 내 프로필, 설정, 구분선, 로그아웃 (180px, radius 8px, shadow-md)
- 콘텐츠: max-width 1080px, 중앙 정렬
- 푸터: #F9FAFB 배경, BoardLog+설명 / 서비스 링크 / 지원 링크 / © 2026 BoardLog

규칙:
- 모바일: 탭바 있음, 푸터 없음
- 데스크톱: 탭바 없음, 푸터 있음
- 비로그인 시 내기록/프로필 탭 → 로그인 유도 (🔒 + "로그인이 필요한 기능이에요" + [로그인] [회원가입])
```

```
- "인기 게임" 22px Bold + "전체 보기 >" 우측

- 데스크톱: 4열 그리드, gap 24px
- GameCard 사용 (썸네일+게임명+★평점+가격)

모든 더미 데이터 한국어. 가격은 "32,000원" 형식.
```

## 프롬프트 7 — 플레이 로그 작성


## 사용 팁

1. **순서대로** — 1→2→3→...→13. 앞에서 만든 컴포넌트를 뒤에서 재사용
2. **한 번에 하나** — 결과 확인 → 수정 → 다음 프롬프트
3. **수정 요청은 구체적으로** — "게임 카드 가격을 16px로", "배너 높이를 좀 더 낮게"
4. **페이지 연결** — 프롬프트 완료 후 "게임 카드 클릭 → /games/:id 이동 연결해줘" 추가 요청
5. **한국어 확인** — 영어가 섞여 나오면 "모든 텍스트를 한국어로 변경해줘" 추가
6. **더미 데이터** — 카탄, 스플렌더, 윙스팬, 아줄, 코드네임, 티켓투라이드, 루트, 팬데믹, 도미니언, 7원더스, 테라포밍 마스, 푸에르토리코
