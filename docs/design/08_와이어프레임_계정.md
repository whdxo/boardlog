### 유효성 검사

| 필드 | 조건 | 에러 메시지 |
|------|------|-----------|
| 이메일 | 이메일 형식, 중복 체크 | "이미 가입된 이메일입니다" |
| 비밀번호 | 8자 이상 | "비밀번호는 8자 이상이어야 합니다" |
| 비밀번호 확인 | 비밀번호와 일치 | "비밀번호가 일치하지 않습니다" |
| 닉네임 | 2~12자, 중복 체크 | "이미 사용 중인 닉네임입니다" |
| 약관 동의 | 체크 필수 | "약관에 동의해주세요" |

**닉네임 중복 체크:** 입력 후 500ms 디바운스로 실시간 확인, ✓ 초록 / ✕ 빨강

**가입 완료 시:** 홈(H-1)으로 이동 + 토스트 "환영합니다!"

## 14. 모바일 ↔ 데스크톱 차이 정리

| 요소 | 모바일 | 데스크톱 |

| 설정 | 전체 화면 리스트 | 좌측 사이드바 + 우측 콘텐츠 |
| 알림 | 전체 화면 | 드롭다운 + 전체 페이지 |
| 프로필 수정 | 전체 화면 | 중앙 카드 (max 500px) |
