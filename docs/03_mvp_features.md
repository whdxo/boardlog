# MVP 기능 정의

> Phase 1에서 반드시 있어야 하는 것만 정의한다.
> "있으면 좋겠다"는 Phase 2 이후로 미룬다.

---

## MVP 포함 기능

### 1. 회원 시스템
- [x] 이메일 회원가입 / 로그인
- [x] 소셜 로그인 (카카오 or 구글)
- [x] 프로필 기본 정보 (닉네임, 프로필 이미지)

### 2. 보드게임 탐색
- [x] 게임 목록 (카드형 UI)
- [x] 게임 상세 페이지
- [x] 키워드 검색
- [x] 기본 필터 (인원수, 시간)
- [x] 게임 평점 등록

### 3. 커뮤니티
- [x] 게시글 작성 (텍스트 + 이미지)
- [x] 게시글 목록 / 상세
- [x] 댓글
- [x] 좋아요
- [x] 태그 (카테고리 대체)

### 4. 내 기록
- [x] 내 컬렉션 등록 (보유 / 위시리스트)
- [x] 플레이 로그 작성
- [x] 내 평점 관리

### 5. 펀딩 (간단 버전)
- [x] 펀딩 프로젝트 목록
- [x] 프로젝트 상세 페이지
- [x] 후원하기 버튼 (외부 링크 or 자체 결제)

---

## MVP 제외 기능 (Phase 2+)

- 팔로우 / 팔로워 피드
- 추천 알고리즘
- 중고 거래 마켓
- 쪽지 / DM
- 오프라인 모임
- 제작자 대시보드 (펀딩 관리)
- 플레이 통계 분석

---

## 우선순위 매트릭스

| 기능 | 중요도 | 개발 난이도 | 우선순위 |
|-----|--------|------------|---------|
| 게임 탐색 & 상세 | 높음 | 중간 | ★★★ |
| 내 기록 | 높음 | 낮음 | ★★★ |
| 커뮤니티 게시판 | 높음 | 중간 | ★★★ |
| 회원 시스템 | 필수 | 낮음 | ★★★ |
| 펀딩 리스트 | 중간 | 낮음 | ★★ |
| 게임 평점/리뷰 | 중간 | 낮음 | ★★ |
| 검색 & 필터 | 중간 | 중간 | ★★ |
| 알림 | 낮음 | 중간 | ★ |
