# MVP 기능 정의

> Phase 1에서 반드시 있어야 하는 것만 정의한다.
> "있으면 좋겠다"는 Phase 2 이후로 미룬다.

---

## MVP 포함 기능

### 1. 회원 시스템
- [x] 이메일 회원가입 / 로그인
- [x] 소셜 로그인 (카카오, Apple, 구글)
- [x] 프로필 기본 정보 (닉네임, 프로필 이미지)

### 2. 보드게임 탐색
- [x] 게임 목록 (카드형 UI)
- [x] 게임 상세 페이지
- [x] 키워드 검색
- [x] 기본 필터 (인원수, 시간, 가격대)
- [x] 게임 평점 등록 (별점)

### 3. 구매
- [x] 게임 상세 페이지에 구매 링크 (외부 쇼핑몰 — 네이버 쇼핑 등)
- [x] 가격 정보 표시 (외부 최저가 등)

### 4. 내 기록
- [x] 내 컬렉션 등록 (보유 / 위시리스트)
- [x] 플레이 로그 작성 (날짜, 함께한 사람, 장소, 메모)
- [x] 내 평점 관리

### 5. 기타
- [x] 북마크 (게임 스크랩)
- [x] 알림 (기본)

---

## MVP 제외 기능 (Phase 2+)

- 자체 결제 / 장바구니 (현재는 외부 링크만)
- 텍스트 리뷰 (현재는 별점만)
- 커뮤니티 (게시판, 댓글, 좋아요, 신고)
- 추천 알고리즘
- 중고 거래 마켓
- 팔로우 / 팔로워 피드
- 쪽지 / DM
- 플레이 통계 분석
- 펀딩 (Phase 3)

---

## 우선순위 매트릭스

| 기능 | 중요도 | 개발 난이도 | 우선순위 |
|-----|--------|------------|---------|
| 게임 탐색 & 상세 | 높음 | 중간 | ★★★ |
| 구매 링크 연결 | 높음 | 낮음 | ★★★ |
| 내 기록 (컬렉션 + 플레이 로그) | 높음 | 낮음 | ★★★ |
| 회원 시스템 | 필수 | 낮음 | ★★★ |
| 게임 평점 (별점) | 중간 | 낮음 | ★★ |
| 검색 & 필터 | 중간 | 중간 | ★★ |
| 북마크 | 중간 | 낮음 | ★★ |
| 알림 | 낮음 | 중간 | ★ |
