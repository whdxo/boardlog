# 서비스 컨셉 & 방향성

## 서비스명 (가칭)
BoardLog

## 한 줄 정의
> 보드게임을 발견하고, 후기를 읽고, 바로 구매할 수 있는 보드게임 종합 플랫폼

---

## 핵심 방향

보드라이프(boardlife.co.kr)를 레퍼런스로 하되, **그대로 복제하지 않음**

| 보드라이프 | BoardLog |
|-----------|---------|
| 커뮤니티 + DB + 거래 + 개인관리 | **구매(스토어)** + 탐색 + 기록 + 리뷰 |
| 올드한 게시판 구조 | 모바일 중심 깔끔한 UX |
| 유저 참여형 DB (정확도 낮음) | 큐레이션 + BGG 연동 |
| 기능 분산, 복잡 | 핵심 기능 집중, 단계적 확장 |

---

## 서비스 성장 로드맵

### Phase 1 — MVP (핵심만)
- 보드게임 탐색 & 정보
- 구매 연결 (외부 쇼핑몰 링크 — 네이버 쇼핑 등)
- 내 게임 기록 (컬렉션, 플레이 로그)
- 평점 (별점만)
- 북마크

### Phase 2 — 확장
- 자체 결제 시스템 (토스페이먼츠 등)
- 텍스트 리뷰 기능
- 커뮤니티 (게시판 + 댓글)
- 추천 시스템
- 중고 거래 마켓

### Phase 3 — 성숙
- 펀딩 (보드게임 프로젝트 후원)
- 오프라인 모임 연결
- 제작자 ↔ 유저 직접 소통 채널
- 데이터 기반 취향 분석

---

## 핵심 가치 (왜 이 서비스인가)

1. **발견** — 내 취향에 맞는 보드게임을 쉽게 찾는다
2. **구매** — 마음에 드는 게임을 바로 살 수 있다
3. **기록** — 플레이한 게임을 나만의 로그로 남긴다
4. **공유** — 다른 사람의 후기와 평점을 참고해 더 좋은 선택을 한다
