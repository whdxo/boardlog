## 빠른 요약

**서비스**: 보드게임 구매 + 탐색 + 기록 플랫폼

**Phase 1 (MVP)**
- 보드게임 탐색 & 정보
- 구매 연결 (외부 쇼핑몰 링크)
- 내 게임 기록 (컬렉션, 플레이 로그 — 누구와, 언제, 어디서)
- 평점 (별점만, 텍스트 리뷰는 Phase 2)

**레퍼런스**: [보드라이프](https://boardlife.co.kr) — 구조 참고, UX는 완전 재설계
