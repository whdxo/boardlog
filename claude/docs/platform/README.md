## 핵심 서비스 맵

```
BoardLog
 ├── 🏠 홈                — 큐레이션, 랭킹, 배너
 ├── 🎲 보드게임          — 탐색, 랭킹, 신작, 게임 상세
 ├── 💬 커뮤니티          — 후기, 정보, 공략, 자유 게시판
 ├── 🤝 모임             — 모임 찾기, 만들기
 ├── 📍 로컬             — 카페, 매장 지도 탐색
 ├── 🛒 쇼핑             — 신규, 인기, 할인
 ├── ♻️  중고거래         — 판매/구매/교환 + 시세비교
 └── 👤 내 게임           — 컬렉션(8종), 기록, 평점, 셀렉션
```

## 차별화 포인트 vs boardlife

| 항목 | boardlife | BoardLog |
|------|-----------|----------|
| 디자인 | 구형 PC 중심 | 모던 Mobile-first |
| 컬렉션 | 8종 (동일) | 8종 + 셀렉션 공유 강화 |
| 플레이 기록 | 기본 | 점수 기록 + 통계 대시보드 |
| 커뮤니티 | 게시판 중심 | 피드 + 팔로우 소셜 피드 |
| 검색 | 단순 텍스트 | 통합 검색 + 자동완성 |
| 모바일 | 반응형 (미흡) | Mobile-first 완전 최적화 |
