# 유저 플로우 (User Flow)

---

## 1. 인증 플로우

### 1-1. 회원가입
```
랜딩/홈
  └─> [회원가입] 클릭
        └─> /signup
              ├─> 이메일 입력 → 인증코드 발송 → 코드 확인 → 비번 설정 → 닉네임 → 완료 → 홈(/)
              └─> [Google로 계속] → Google OAuth → 닉네임 설정 → 완료 → 홈(/)
```

### 1-2. 로그인
```
어느 페이지에서든 [로그인 필요] 액션 시도
  └─> /login
        ├─> 이메일 + 비번 → 홈 또는 이전 페이지로 복귀
        ├─> [Google로 로그인] → OAuth → 복귀
        └─> [비밀번호 찾기] → /forgot-password → 이메일 발송 → /reset-password
```

### 1-3. 로그인 필요 액션 처리
```
비로그인 상태에서 액션 시도 (평점/컬렉션/글작성 등)
  └─> 로그인 유도 바텀시트 등장
        ├─> [로그인] → /login?callbackUrl=현재URL
        └─> [취소] → 시트 닫기
```

## 2. 게임 탐색 플로우

### 2-1. 홈 → 게임 발견
```
홈(/)
  ├─> 실시간 랭킹 게임 클릭 → /games/[id]
  ├─> 큐레이션 섹션 [더보기] → /games
  └─> 검색바 클릭 → /search
```

### 2-2. 게임 탐색 → 상세
```
/games
  ├─> 필터 칩 선택 (인원/장르/가격/시간)
  ├─> 정렬 변경 (인기/평점/신규)
  ├─> 검색어 입력
  └─> 게임 카드 클릭 → /games/[id]
```

### 2-3. 게임 상세 → 액션
```
/games/[id]  (정보 탭 기본)
  │
  ├─> [컬렉션 추가] 클릭
  │     ├─> 비로그인: 로그인 유도 시트
  │     └─> 로그인: 상태 선택 시트 (보유/팬/위시/...) → 저장 → 토스트
  │
  ├─> [별점 평가] 클릭
  │     ├─> 비로그인: 로그인 유도 시트
  │     └─> 로그인: 평점 모달 → 별점 + 한줄평 → 저장 → 토스트
  │
  ├─> [플레이 기록] 탭 → /games/[id]?tab=plays
  │     └─> [기록 추가] → /my/plays/write?gameId=[id]
  │
  ├─> [구매] 탭 → 외부 쇼핑몰 링크 (새 탭)
  │
  └─> [커뮤니티] 탭 → 관련 게시글 목록
        └─> 게시글 클릭 → /community/[id]
```

---

## 3. 플레이 기록 작성 플로우

```
진입점 (여러 곳)
  ├─> /games/[id] [기록 추가] 버튼 (게임 사전 선택)
  ├─> /my/plays [기록 작성] FAB
  └─> 탭바 [내 게임] → [기록] 탭 → [+]

/my/plays/write
  1. 게임 선택 (사전 선택 아닌 경우 → 게임 검색 시트)
  2. 날짜 선택 (기본: 오늘)
  3. 함께한 사람 입력 (chip 방식)
  4. 장소 입력 (텍스트 or 로컬 연동)
  5. 별점 (선택)
  6. 점수 기록 (선택)
  7. 메모 (선택)
  8. 사진 (선택, 최대 5장)
  └─> [저장] → /my/plays → 토스트
```

## 4. 커뮤니티 플로우

### 4-1. 글 읽기 (비로그인 가능)
```
/community (피드)
  ├─> 탭 전환: 최신 / 베스트 / 팔로우
  ├─> 게시판 필터: 후기 / 정보 / 공략 / 자유
  └─> 게시글 카드 클릭 → /community/[id]
        ├─> 댓글 읽기 (비로그인 가능)
        ├─> [댓글 작성] → 비로그인: 로그인 유도
        └─> [추천] → 비로그인: 로그인 유도
```

### 4-2. 글 작성 (로그인 필요)
```
[글쓰기] 버튼 클릭
  ├─> 비로그인: 로그인 유도
  └─> 로그인: /community/new
        1. 게시판 선택 (후기/정보/공략/자유)
        2. 제목 입력
        3. 본문 입력 (텍스트 + 이미지 업로드)
        4. 게임 태그 추가 (선택)
        └─> [발행] → /community/[id] → 토스트
```

### 4-3. 댓글 작성
```
/community/[id]
  └─> 댓글 입력창 클릭
        ├─> 비로그인: 로그인 유도 토스트
        └─> 로그인: 텍스트 입력 → [등록] → 댓글 추가
```

## 5. 로컬 탐색 플로우

```
/local
  ├─> 지도 뷰 (기본)
  │     ├─> 현재 위치 허용 → 내 위치 중심 지도
  │     ├─> 마커 클릭 → 장소 미리보기 카드 (하단)
  │     └─> 미리보기 카드 클릭 → /local/[id]
  │
  └─> 리스트 뷰 전환
        ├─> 지역 필터 (시/도 → 구/군)
        ├─> 카테고리 필터 (카페/매장)
        └─> 장소 카드 클릭 → /local/[id]

/local/[id]
  ├─> 정보 탭 (운영시간, 연락처, 지도)
  ├─> 보유 게임 탭 → 게임 클릭 → /games/[id]
  └─> 리뷰 탭
        ├─> 비로그인: 읽기만
        └─> 로그인: [리뷰 작성] → 별점 + 텍스트 → 저장
```

## 6. 쇼핑 플로우

```
/shop
  ├─> 필터 (카테고리/가격/인원/시간)
  ├─> 탭: 전체 / 신규 / 인기 / 할인
  └─> 상품 카드 클릭 → /shop/[id]
        ├─> 상품 정보 + 가격
        ├─> [위시리스트 추가] → 로그인 필요
        ├─> [가격 변동 알림] → 로그인 필요
        └─> [구매하러 가기] → 외부 쇼핑몰 (새 탭)
```

## 7. 중고거래 플로우

### 7-1. 목록 탐색 (비로그인 가능)
```
/used
  ├─> 탭: 전체 / 판매 / 구매 / 교환 / 나눔
  ├─> 게임 검색 필터
  └─> 거래글 클릭 → /used/[id]
        ├─> 내용 + 사진 확인
        ├─> [댓글로 문의] → 비로그인: 로그인 유도
        └─> 게임 태그 클릭 → /games/[id]
```

### 7-2. 글 작성 (로그인 필요)
```
[거래글 쓰기] 클릭
  ├─> 비로그인: 로그인 유도
  └─> 로그인: /used/write
        1. 거래 유형 (판매/구매/교환/나눔)
        2. 게임 태그 (검색으로 추가)
        3. 가격 입력 (나눔이면 0)
        4. 상태 선택 (미개봉/상/중/하)
        5. 거래 방식 (택배/직거래)
        6. 사진 업로드 (최대 5장)
        7. 본문
        └─> [등록] → /used/[id] → 토스트
```

### 7-3. 거래 완료 처리
```
/used/[id] (내 게시글)
  └─> [거래 완료] 버튼 → 확인 다이얼로그 → 상태 변경 → 뱃지 "완료" 표시
```

## 8. 내 게임 플로우

```
/my
  ├─> 컬렉션 탭
  │     ├─> 상태별 탭 (보유/팬/위시/...)
  │     └─> 게임 카드 클릭 → /games/[id]
  │
  ├─> 기록 탭
  │     ├─> 월별 그룹 리스트
  │     ├─> 기록 카드 클릭 → /my/plays/[id]
  │     └─> [+] FAB → /my/plays/write
  │
  ├─> 평점 탭
  │     └─> 게임별 내 별점 목록 → 클릭 → 평점 수정 모달
  │
  └─> 셀렉션 탭
        ├─> 셀렉션 카드 클릭 → /my/selections/[id]
        └─> [셀렉션 만들기] → 이름/설명 입력 → 게임 추가 → 저장
```

---

## 9. 검색 플로우

```
GNB 검색 아이콘 or 모바일 탐색탭 클릭 → /search
  └─> 검색어 입력 (자동완성 드롭다운)
        └─> Enter or 검색어 클릭
              └─> 결과 탭: 게임 / 게시글 / 유저 / 로컬
                    └─> 각 결과 클릭 → 해당 상세 페이지
```

---

## 10. 프로필/팔로우 플로우

```
게시글 작성자 클릭 or 유저 검색 → /profile/[id]
  ├─> 게임 컬렉션 (공개 설정된 경우)
  ├─> 작성한 게시글
  └─> [팔로우] 버튼
        ├─> 비로그인: 로그인 유도
        └─> 로그인: 팔로우 → 팔로우 피드에 해당 유저 활동 노출
```
