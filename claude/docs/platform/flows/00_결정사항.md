# 확정 스펙 결정사항

| 항목 | 결정 | 비고 |
|------|------|------|
| 비로그인 접근 | 열람 전체 허용 | 작성/컬렉션/평점 등 액션만 로그인 필요 |
| 온보딩 | 없음 | 가입 후 바로 홈 |
| 소셜 로그인 | Google + 이메일 우선 구현 | Kakao/Naver/Apple은 UI 틀만, 추후 연동 |
| 커뮤니티 에디터 | 심플 (텍스트 + 이미지 업로드) | 리치텍스트 X |
| 쇼핑 결제 | 외부 쇼핑몰 링크 연결 | 자체 결제 없음 |
| 중고 채팅 | 댓글 방식 | 추후 1:1 채팅으로 업그레이드 |
| 지도 API | 카카오맵 | 데이터 품질 + API 안정성 |
