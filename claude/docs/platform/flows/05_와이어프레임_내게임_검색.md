## 인증 — 로그인 (/login)

```
┌─────────────────────────┐
│         BoardLog         │
├─────────────────────────┤
│                          │
│       로그인             │
│                          │
│ [──────────────────────]│  ← 이메일
│ [──────────────────────]│  ← 비밀번호
│                  [로그인]│
│                          │
│ ─────────── 또는 ────────│
│                          │
│ [G  Google로 계속하기  ] │
│ [K  카카오 (준비중)    ] │  ← 비활성화
│ [N  네이버 (준비중)    ] │  ← 비활성화
│                          │
│ 계정이 없으신가요? [회원가입]│
│        [비밀번호 찾기]    │
└─────────────────────────┘
```
