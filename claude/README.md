# Claude Code 작업 관리

## 폴더 구조

```
claude/
├── docs/          ← 프로젝트 설계/참고 문서
├── in-progress/   ← 현재 작업 중
├── done/          ← 완료된 작업
└── archive/       ← 과거 문서 보관
```

## 작업 흐름

```
in-progress → done → archive (필요 시)
```

- `in-progress/` : 지금 하고 있는 작업 문서를 여기에 작성
- `done/` : 끝난 작업은 여기로 이동
- `archive/` : 오래된 문서 보관
- `docs/` : API, DB, 플로우 등 설계 참고 문서
