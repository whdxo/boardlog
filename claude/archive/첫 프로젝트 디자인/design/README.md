## 작업 순서

```
1. 디자인 원칙 & 레퍼런스 분석
   ↓
2. 디자인 시스템 (컬러, 타이포, 컴포넌트)
   ↓
3. 전체 화면 목록 & 흐름도
   ↓
4. 공통 레이아웃 와이어프레임
   ↓
5~8. 각 섹션별 와이어프레임 (홈, 탐색, 내기록, 계정)
   ↓
9. 인터랙션 & 마이크로 UX 정의
   ↓
피그마 시안 제작
```
