## 3. 간격 & 그리드

### Spacing Scale (4px 기반)

```
xs:  4px    — 아이콘과 텍스트 사이
sm:  8px    — 인라인 요소 간격
md:  12px   — 관련 요소 그룹 내부
base: 16px  — 카드 패딩, 리스트 아이템 간격
lg:  24px   — 카드 간 간격, 섹션 내 그룹
xl:  32px   — 섹션 간 간격 (모바일)
2xl: 48px   — 섹션 간 간격 (데스크톱)
```

### Grid

```
Mobile:   1열, 좌우 패딩 16px
Tablet:   2열, gap 16px, 좌우 패딩 24px
Desktop:  최대 1080px, 4열, gap 24px, 중앙 정렬
```

## 4. 컴포넌트

### 4.1 버튼

```
[  Primary Button  ]     ← primary-500 배경, 흰 텍스트, radius 8px
[  Secondary Button ]    ← 흰 배경, primary-500 보더+텍스트
[  Ghost Button  ]       ← 배경 없음, primary-500 텍스트
[  Danger Button ]       ← error 배경, 흰 텍스트 (삭제 등)

크기:
  sm: 높이 32px, 14px 텍스트, 패딩 12px
  md: 높이 40px, 14px 텍스트, 패딩 16px
  lg: 높이 48px, 16px 텍스트, 패딩 20px (CTA)
```

### 4.2 게임 카드

게임 목록에서 사용하는 핵심 컴포넌트.

```
┌─────────────────────┐
│                     │
│    [게임 썸네일]      │  ← 3:4 비율, radius 8px
│     이미지 영역       │
│                     │
├─────────────────────┤
│ 게임 제목             │  ← h2 (18px, SemiBold)
│ ★ 4.2  ·  2~4인     │  ← caption (12px), 별 accent-400
│ 35,000원            │  ← body (14px, SemiBold), primary-500
└─────────────────────┘

hover: translateY(-2px), shadow 증가
```

**카드에 표시하는 정보:**
- 썸네일 (필수)
- 게임명 (필수)
- 평점 (★ 숫자)
- 인원수 (2~4인)
- 가격 (대표 최저가)

> 플레이 시간, 연령, 제작사, 출시년도 등은 상세 페이지에서만 표시

### 4.3 게시글 카드 (커뮤니티 피드)

이미지가 있는 경우와 없는 경우 2가지.

```
이미지 있는 게시글:
┌─────────────────────────────────┐
│ 👤 닉네임  ·  2시간 전    [후기]  │  ← 태그는 pill 뱃지
├─────────────────────────────────┤
│ 게시글 제목                       │  ← h2
│ 본문 미리보기 2줄까지...           │  ← body, gray-600
├─────────────────────────────────┤
│                                 │
│       [이미지 프리뷰]             │  ← 16:9 비율, radius 8px
│                                 │
├─────────────────────────────────┤
│ ♡ 12   💬 3   🔖                │  ← 좋아요, 댓글수, 북마크
└─────────────────────────────────┘

텍스트만 있는 게시글:
┌─────────────────────────────────┐
│ 👤 닉네임  ·  2시간 전    [질문]  │
├─────────────────────────────────┤
│ 게시글 제목                       │
│ 본문 미리보기 2줄까지...           │
├─────────────────────────────────┤
│ ♡ 12   💬 3   🔖                │
└─────────────────────────────────┘
```

**피드 스타일:** 이미지가 크게 보이는 인스타 스타일 기반, 텍스트 전용 글도 자연스럽게 섞임

### 4.4 별점

```
★ ★ ★ ★ ☆     4.0 / 5.0

- 5점 만점, 1점 단위 (반개 없음)
- 채워진 별: accent-400
- 빈 별: gray-200
- 터치/클릭으로 선택
- 선택 시 살짝 스케일 애니메이션 (scale 1.1 → 1.0)
```

### 4.5 태그 / 뱃지

```
게시글 태그:
  [후기] [질문] [추천] [정보] [자유]
  ← pill 형태 (radius 9999px)
  ← primary-100 배경, primary-600 텍스트
  ← 12px, 패딩 4px 10px

게임 상태 뱃지:
  [보유]       ← success 배경, 흰 텍스트
  [위시리스트]  ← accent-100 배경, accent-700 텍스트
  [플레이완료]  ← gray-100 배경, gray-600 텍스트

장르 태그:
  [전략] [가족] [파티] [추리]
  ← gray-100 배경, gray-700 텍스트
```

### 4.6 인풋

```
┌─────────────────────────────┐
│ 플레이스홀더 텍스트            │
└─────────────────────────────┘

기본:    gray-200 보더, white 배경
포커스:  primary-500 보더, primary-50 배경
에러:    error 보더, error 텍스트 아래 표시

높이: 40px, radius 8px, 패딩 12px
```

### 4.7 모달 / 바텀시트

```
Desktop: 중앙 모달, 최대 480px, radius 12px
Mobile:  바텀시트 (하단에서 올라옴), radius 상단 16px

배경: 반투명 오버레이 (black 50%)
```

### 4.8 플레이 로그 입력 폼

```
MVP:
┌─────────────────────────────┐
│  📅 플레이 날짜  [2026-04-01] │
│  🎲 게임 선택    [검색...]    │
│  👥 함께한 사람  [이름 추가...]│
│  📍 장소        [입력...]    │
│  ★ ★ ★ ★ ☆   별점          │
│  📝 메모 (선택)              │
│  ┌─────────────────────┐    │
│  │                     │    │
│  └─────────────────────┘    │
│                             │
│  [ 기록 저장 ]               │  ← Primary 버튼
└─────────────────────────────┘

Phase 2 추가 필드 (예정):
  🏆 승패 / 순위 기록
  ⏱ 플레이 시간 (분)
  📸 사진 첨부
```

### 4.9 알림 아이템

```
┌─────────────────────────────────────┐
│ 🔵 👤 닉네임님이 내 게시글에 댓글을 남겼습니다  │
│    "댓글 미리보기..."  ·  3분 전            │
├─────────────────────────────────────┤
│    👤 닉네임님이 내 게시글을 좋아합니다        │
│    게시글 제목  ·  1시간 전                  │
└─────────────────────────────────────┘

읽지 않은 알림: 좌측 파란 점(🔵), primary-50 배경
읽은 알림: 배경 없음
```

### 4.10 빈 상태 (Empty State)

```
┌─────────────────────────────┐
│                             │
│         🎲                  │
│   아직 기록이 없어요          │
│   첫 플레이를 기록해볼까요?    │  ← gray-400 텍스트
│                             │
│   [ 기록하기 ]               │  ← Secondary 버튼
│                             │
└─────────────────────────────┘
```

## 7. 애니메이션

```
기본 트랜지션:  150ms ease-in-out   ← hover, 색상 변경
카드 hover:    200ms ease-out      ← translateY(-2px)
모달 진입:     250ms ease-out      ← fadeIn + slideUp
바텀시트:      300ms ease-out      ← slideUp
별점 선택:     200ms spring        ← scale(1.1) → scale(1.0)
```

> 모든 애니메이션은 `prefers-reduced-motion` 미디어 쿼리 존중
