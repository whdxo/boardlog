# 와이어프레임 — 공통 레이아웃

> 모든 화면에서 반복되는 공통 구조 정의

---

## 1. 모바일 레이아웃 (~ 767px)

```
┌─────────────────────────────┐
│  상단 헤더 (56px)             │
├─────────────────────────────┤
│                             │
│                             │
│                             │
│        콘텐츠 영역            │
│     (스크롤 가능, flex-1)     │
│                             │
│                             │
│                             │
├─────────────────────────────┤
│  하단 탭바 (56px)             │
└─────────────────────────────┘
```

### 모바일 상단 헤더

```
┌─────────────────────────────────────┐
│  BoardLog          🔔(알림)  ⚙️(설정) │
│  [로고/텍스트]                        │
└─────────────────────────────────────┘

상세 페이지 진입 시:
┌─────────────────────────────────────┐
│  ← (뒤로)    페이지 제목     ⋮(더보기) │
└─────────────────────────────────────┘
```

| 요소 | 규격 |
|------|------|
| 높이 | 56px |
| 좌우 패딩 | 16px |
| 로고 | 텍스트형 "BoardLog", h2 (18px Bold), primary-500 |
| 뒤로가기 | `ChevronLeft` 아이콘, 24px |
| 우측 액션 | 아이콘 20px, 간격 12px |
| 배경 | white, 하단 보더 gray-200 (1px) |

### 모바일 하단 탭바

```
┌─────────────────────────────────────┐
│  🏠      🎲       📚       👤      │
│  홈      탐색     내기록    프로필    │
└─────────────────────────────────────┘
```

| 요소 | 규격 |
|------|------|
| 높이 | 56px + safe area (iOS 하단 여백) |
| 배경 | white, 상단 보더 gray-200 (1px) |
| 아이콘 | 24px, 비활성 gray-400 / 활성 primary-500 |
| 텍스트 | 11px Medium, 비활성 gray-400 / 활성 primary-500 |
| 아이콘-텍스트 간격 | 2px |
| 탭 영역 | 균등 분할 (25% × 4) |

**탭바 동작:**
- 현재 탭: 아이콘 filled + primary-500 컬러
- 비활성 탭: 아이콘 outline + gray-400
- 알림 뱃지: 빨간 점 (8px) 아이콘 우상단 (알림 있을 때)
- 스크롤 시 탭바 항상 고정

## 2. 데스크톱 레이아웃 (1024px ~)

```
┌───────────────────────────────────────────────────┐
│                    상단 GNB (64px)                  │
├───────────────────────────────────────────────────┤
│                                                   │
│              ┌─────────────────────┐              │
│              │                     │              │
│              │    콘텐츠 영역        │              │
│              │   (max 1080px)      │              │
│              │    중앙 정렬          │              │
│              │                     │              │
│              └─────────────────────┘              │
│                                                   │
├───────────────────────────────────────────────────┤
│                    푸터                             │
└───────────────────────────────────────────────────┘
```

### 데스크톱 상단 GNB

```
┌──────────────────────────────────────────────────────────────┐
│  BoardLog       홈    탐색    내기록          🔔  👤 닉네임 ▾  │
│  [로고]       [메뉴 중앙 배치]            [우측: 알림 + 프로필] │
└──────────────────────────────────────────────────────────────┘
```

| 요소 | 규격 |
|------|------|
| 높이 | 64px |
| 최대 너비 | 1080px (중앙 정렬) |
| 좌우 패딩 | 24px |
| 로고 | 텍스트형 "BoardLog", h1 (22px Bold), primary-500 |
| 메뉴 | body (14px Medium), 간격 32px |
| 활성 메뉴 | primary-500 텍스트 + 하단 2px 바 |
| 비활성 메뉴 | gray-600 텍스트 |
| 프로필 | 아바타(32px 원형) + 닉네임 + 드롭다운 화살표 |
| 배경 | white, 하단 그림자 shadow-sm |

**프로필 드롭다운:**
```
┌──────────────────┐
│  👤 내 프로필      │
│  ⚙️ 설정          │
│  ───────────────  │
│  🚪 로그아웃      │
└──────────────────┘

너비: 180px
radius: 8px
shadow: shadow-md
```

**비로그인 상태:**
```
┌──────────────────────────────────────────────────────────────┐
│  BoardLog       홈    탐색              [로그인] [회원가입]    │
└──────────────────────────────────────────────────────────────┘

[로그인]: Ghost 버튼
[회원가입]: Primary 버튼 (sm)
```

## 5. 페이지 전환 패턴

### 모바일

| 전환 | 애니메이션 |
|------|-----------|
| 탭 간 이동 | 페이드 (150ms) |
| 목록 → 상세 | 우측에서 슬라이드 인 (250ms) |
| 상세 → 뒤로 | 좌측으로 슬라이드 아웃 (250ms) |
| 모달 열기 | 오버레이 페이드 + 중앙 스케일업 (200ms) |
| 바텀시트 열기 | 하단에서 슬라이드업 (300ms) |

### 데스크톱

| 전환 | 애니메이션 |
|------|-----------|
| 페이지 이동 | 페이드 (150ms) |
| 모달 열기 | 오버레이 페이드 + 중앙 스케일업 (200ms) |

## 6. 스크롤 동작

| 요소 | 모바일 | 데스크톱 |
|------|--------|---------|
| 상단 헤더 | 고정 (항상 보임) | 고정 |
| 하단 탭바 | 고정 (항상 보임) | 없음 |
| 목록 스크롤 | 무한 스크롤 (페이지네이션 X) | 무한 스크롤 |
| 맨 위로 버튼 | 일정 이상 스크롤 시 우하단에 FAB 표시 | 동일 |

### 맨 위로 버튼

```
     ┌─────┐
     │  ↑  │  40px × 40px 원형
     └─────┘  primary-500 배경, 흰 화살표
              shadow-md
              하단 탭바 위 16px
              스크롤 200px 이상 시 fadeIn
```

## 8. 비로그인 vs 로그인 차이

| 요소 | 비로그인 | 로그인 |
|------|---------|--------|
| 홈 | 인기 게임만 표시 | 인기 게임 + 내 최근 기록 |
| 탐색 | 게임 목록/상세 열람 가능 | + 평점 등록, 컬렉션 추가, 북마크 |
| 내기록 탭 | 탭 클릭 시 로그인 유도 | 정상 접근 |
| 프로필 탭 | 탭 클릭 시 로그인 화면 | 정상 접근 |
| GNB 우측 | [로그인] [회원가입] 버튼 | 알림 + 프로필 드롭다운 |
| 탭바 | 동일 4탭 (내기록/프로필 탭 시 로그인 유도) | 동일 4탭 |

### 로그인 유도 화면 (인라인)

```
┌─────────────────────────────┐
│                             │
│         🔒                  │
│   로그인이 필요한 기능이에요    │
│   로그인하고 기록을 시작하세요  │
│                             │
│   [ 로그인 ]  [ 회원가입 ]    │
│                             │
└─────────────────────────────┘

내기록/프로필 탭 진입 시 콘텐츠 영역에 표시
```

## 9. 토스트 메시지

```
모바일 (하단, 탭바 위):
┌─────────────────────────────┐
│  ✓ 저장되었습니다              │  ← success
└─────────────────────────────┘

┌─────────────────────────────┐
│  ✕ 오류가 발생했습니다         │  ← error
└─────────────────────────────┘

데스크톱 (우상단):
                    ┌──────────────────────┐
                    │  ✓ 저장되었습니다       │
                    └──────────────────────┘
```

| 요소 | 규격 |
|------|------|
| 위치 | 모바일: 하단 탭바 위 16px / 데스크톱: 우상단 24px |
| 배경 | gray-900 (90% 불투명) |
| 텍스트 | white, 14px |
| 아이콘 | 좌측 16px, success 초록 / error 빨강 |
| radius | 8px |
| 노출 시간 | 3초 → fadeOut |
| 최대 너비 | 모바일 100% - 32px / 데스크톱 360px |
