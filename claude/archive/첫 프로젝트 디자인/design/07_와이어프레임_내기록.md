> 내가 가진 게임, 플레이 기록, 평점, 북마크를 한 곳에서 관리

## 1. 내 기록 메인 (M-1)

### 모바일

```
┌─────────────────────────────────────┐
│  내 기록                             │  ← 헤더

│  │ 보유    │ │ 플레이  │ │ 평점   │  │
│  └────────┘ └────────┘ └────────┘  │
│                                     │
├─────────────────────────────────────┤
│  [컬렉션]  [로그]  [평점]  [북마크]   │  ← 탭
├─────────────────────────────────────┤
│                                     │
│  (선택한 탭의 리스트 콘텐츠)           │
│                                     │
│                                     │
│                                     │
│                                     │
│                                     │

---

### 통계 카드

```
  ┌────────┐ ┌────────┐ ┌────────┐
  │ 🎲 12  │ │ 📝  8  │ │ ⭐ 15  │
  │ 보유    │ │ 플레이  │ │ 평점   │
  └────────┘ └────────┘ └────────┘
```

| 요소 | 규격 |
|------|------|
| 레이아웃 | 3열 균등 분할 |
| 카드 배경 | gray-50, radius 12px |
| 숫자 | h1 (22px Bold), gray-900 |
| 아이콘 | 16px, 숫자 앞 |
| 라벨 | caption (12px), gray-600 |
| 패딩 | 16px |
| 높이 | 72px |
| 탭 시 | 해당 탭으로 전환 (보유 → 컬렉션, 플레이 → 로그, 평점 → 평점) |

---

### 탭 바

```
  [컬렉션]   [로그]   [평점]   [북마크]
  ━━━━━━━
```

| 요소 | 규격 |
|------|------|
| 높이 | 44px |
| 활성 탭 | primary-500 텍스트, 하단 2px 바 (primary-500) |
| 비활성 탭 | gray-600 텍스트 |
| 텍스트 | body (14px SemiBold) |
| 균등 분할 | 4등분 |
| sticky | 스크롤 시 상단 고정 (헤더 아래) |

## 2. 컬렉션 탭 (M-2)

### 서브 필터

```
  [전체 12]  [보유 8]  [위시 3]  [플레이완료 1]
```

| 요소 | 규격 |
|------|------|
| 형태 | pill 칩, 가로 스크롤 |
| 활성 | primary-100 배경, primary-600 텍스트 |
| 비활성 | gray-100 배경, gray-600 텍스트 |
| 숫자 | 각 상태별 개수 표시 |

### 컬렉션 리스트

```
┌─────────────────────────────────────┐
│  [전체 12]  [보유 8]  [위시 3]  [완료 1] │
├─────────────────────────────────────┤
│                                     │
│  ┌──────┐  카탄                     │
│  │ 썸네일 │  ★ 4.2  ·  2~4인        │
│  │ 60x80 │  32,000원                │
│  └──────┘  [보유] 상태 뱃지          │
│  ─────────────────────────────────  │
│  ┌──────┐  스플렌더                  │
│  │ 썸네일 │  ★ 3.8  ·  2~4인        │
│  │ 60x80 │  28,000원                │
│  └──────┘  [위시리스트] 상태 뱃지     │
│  ─────────────────────────────────  │
│  ┌──────┐  윙스팬                    │
│  │ 썸네일 │  ★ 4.5  ·  1~5인        │
│  │ 60x80 │  45,000원                │
│  └──────┘  [보유] 상태 뱃지          │
│                                     │
└─────────────────────────────────────┘
```

| 요소 | 규격 |
|------|------|
| 썸네일 | 60×80px (3:4), radius 8px |
| 게임명 | h3 (16px SemiBold) |
| 정보 | caption (12px), gray-600 |
| 가격 | body (14px SemiBold), primary-500 |
| 상태 뱃지 | pill, 보유=success / 위시=accent / 완료=gray |
| 아이템 높이 | 88px |
| 구분선 | gray-100 (1px) |
| 탭 시 | 게임 상세(E-2)로 이동 |
| 스와이프 (모바일) | 좌로 스와이프 → [상태 변경] [삭제] 버튼 노출 |

### 빈 상태

```
┌─────────────────────────────────────┐
│                                     │
│              🎲                     │
│    아직 컬렉션이 비어있어요            │
│    게임을 탐색하고 추가해보세요        │
│                                     │
│    [ 게임 탐색하기 ]                  │
│                                     │
└─────────────────────────────────────┘
```

## 3. 로그 탭 (M-3)

```
┌─────────────────────────────────────┐
│                                     │
│  2026년 3월                          │  ← 월별 그룹 헤더

│  3.28 (금)                          │
│  ┌─────────────────────────────┐   │
│  │ 🎲 카탄                      │   │
│  │ 👥 민수, 지영   📍 보드게임카페 │   │
│  │ ★★★★☆       메모 있음 📝    │   │
│  └─────────────────────────────┘   │
│                                     │
│  3.25 (화)                          │
│  ┌─────────────────────────────┐   │
│  │ 🎲 스플렌더                   │   │
│  │ 👥 지영        📍 집          │   │
│  │ ★★★☆☆                      │   │
│  └─────────────────────────────┘   │
│                                     │
│  2026년 2월                          │
│  ─────────────────────────────────  │
│                                     │
│  2.15 (토)                          │
│  ┌─────────────────────────────┐   │
│  │ 🎲 아줄                      │   │
│  │ 👥 민수, 철수   📍 민수네 집   │   │
│  │ ★★★★★       메모 있음 📝    │   │
│  └─────────────────────────────┘   │
│                                     │
│         (무한 스크롤)                 │
│                                     │
├─────────────────────────────────────┤
│           [ + 기록하기 ]             │  ← FAB or 하단 버튼
└─────────────────────────────────────┘
```

| 요소 | 규격 |
|------|------|
| 월별 헤더 | h2 (18px SemiBold), gray-900, sticky |
| 날짜 | body (14px SemiBold), gray-700 |
| 로그 카드 배경 | white, radius 12px, shadow-sm |
| 게임명 | h3 (16px SemiBold) |
| 함께한 사람 | caption (12px), gray-600, 👥 아이콘 |
| 장소 | caption (12px), gray-600, 📍 아이콘 |
| 별점 | 14px, accent-400 |
| 메모 표시 | 📝 아이콘만 (내용은 상세에서) |
| 카드 간격 | 12px |
| 카드 탭 시 | 로그 상세(M-6)로 이동 |
| 기록 버튼 | 모바일: 우하단 FAB (56px 원형, primary-500) / 데스크톱: 상단 우측 Primary 버튼 |

### 기록하기 FAB (모바일)

```
          ┌─────┐
          │  +  │  56px × 56px 원형
          └─────┘  primary-500 배경, 흰 + 아이콘
                   shadow-md
                   하단 탭바 위 16px, 우측 16px
```

### 각 필드 상세

**게임 검색 모달:**
```
┌─────────────────────────────────────┐
│  ── (드래그 핸들)                     │
│                                     │
│  🔍 게임 검색                        │
│  ┌─────────────────────────────┐   │
│  │  게임명을 입력하세요...        │   │
│  └─────────────────────────────┘   │
│                                     │
│  ┌──────┐  카탄                     │
│  │ 썸네일 │  2~4인  ·  60~90분      │
│  └──────┘                           │
│  ─────────────────────────────────  │
│  ┌──────┐  카탄 주니어               │
│  │ 썸네일 │  2~4인  ·  30분          │
│  └──────┘                           │
│  ─────────────────────────────────  │
│  ┌──────┐  카탄 확장판               │
│  │ 썸네일 │  3~4인  ·  90분          │
│  └──────┘                           │
│                                     │
└─────────────────────────────────────┘

최대 높이: 화면 80%
검색: 300ms 디바운스
결과 탭 시: 게임 선택 → 모달 닫힘
```

**함께한 사람 입력:**

| 요소 | 규격 |
|------|------|
| 입력 | 텍스트 인풋, 엔터 시 칩 추가 |
| 칩 | pill 형태, gray-100 배경, ✕ 삭제 버튼 |
| 최대 인원 | 제한 없음 |
| 자동완성 | 이전에 입력한 이름 자동 추천 (Phase 2) |

**장소 입력:**

| 요소 | 규격 |
|------|------|
| 입력 | 텍스트 인풋, 자유 텍스트 |
| 플레이스홀더 | "어디서 했나요?" |
| 자동완성 | 이전에 입력한 장소 자동 추천 (Phase 2) |

**날짜 피커:**

| 요소 | 모바일 | 데스크톱 |
|------|--------|---------|
| 형태 | OS 기본 날짜 피커 | 캘린더 드롭다운 |
| 기본값 | 오늘 날짜 |
| 미래 날짜 | 선택 불가 |

**별점:**

| 요소 | 규격 |
|------|------|
| 크기 | 32px, 간격 8px |
| 선택 | 탭한 별까지 채워짐 |
| 필수 여부 | 선택 사항 (빈 별 상태로 저장 가능) |

**저장 동작:**
- 게임 + 날짜 필수, 나머지 선택
- 저장 시 → 로그 목록(M-3)으로 이동 + 토스트 "기록이 저장되었습니다"
- 헤더 우측 [저장] (모바일) / 하단 [저장하기] 버튼 (데스크톱)

## 5. 플레이 로그 수정 (M-5)

- M-4와 동일한 폼 재사용

- 헤더: "기록 수정"
- 저장 시 → 로그 상세(M-6)로 이동 + 토스트

## 6. 플레이 로그 상세 (M-6)

```

│  ←  기록 상세               ⋮(더보기) │
├─────────────────────────────────────┤
│                                     │
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  │  🎲 카탄                     │   │  ← 게임명 (탭 → E-2)
│  │                             │   │
│  │  ★ ★ ★ ★ ☆               │   │
│  │                             │   │
│  └─────────────────────────────┘   │
│                                     │
│  📅  2026년 3월 28일 (금)            │
│  👥  민수, 지영                      │
│  📍  강남 보드게임 카페               │
│                                     │
│  ─────────────────────────────────  │
│                                     │
│  📝 메모                             │
│  오늘 카탄 3판 했는데 처음으로         │
│  최장도로 가져옴! 민수가 좀 억울해      │
│  했지만 재밌었다                       │
│                                     │
├─────────────────────────────────────┤
│  🏠      🎲       📚       👤      │
│  홈      탐색     내기록    프로필    │
└─────────────────────────────────────┘
```

**더보기(⋮) 메뉴:**
```
┌──────────────┐
│  ✏️ 수정하기   │  → M-5
│  🗑️ 삭제하기  │  → 삭제 확인 모달
└──────────────┘
```

| 요소 | 규격 |
|------|------|
| 게임명 | h1 (22px Bold), 탭 시 게임 상세(E-2) |
| 별점 | 24px, accent-400, 읽기 전용 |
| 정보 | body (14px), gray-700, 아이콘 + 텍스트 |
| 정보 간 간격 | 8px |
| 메모 | body (14px), gray-700, line-height 22px |
| 메모 구분 | 상단 구분선 gray-100 |

### 삭제 확인 모달

```
┌───────────────────────────────┐
│                               │
│  이 기록을 삭제할까요?          │

## 9. 모바일 ↔ 데스크톱 차이 정리

| 요소 | 모바일 | 데스크톱 |
|------|--------|---------|
| 통계 카드 | 3열, 작은 카드 | 3열, 큰 카드 |
| 탭 | 4등분, sticky | 좌측 정렬, 하단 바 |
| 컬렉션 리스트 | 1열 | 2열 |
| 로그 리스트 | 1열 카드 | 1열 카드 (max 800px) |
| 로그 작성 | 전체 화면 | 중앙 카드 (max 600px) |
| 기록하기 버튼 | FAB (우하단) | 상단 우측 Primary 버튼 |
| 컬렉션 삭제 | 좌 스와이프 | 호버 시 삭제 아이콘 |
