|---|------|------|------|

## 3. 화면 흐름도

### 메인 네비게이션 흐름

```
┌──────────────────────────────────────────┐
│           탭바 / GNB (항상 노출)            │
│   [홈]      [탐색]      [내기록]    [프로필] │
└──┬──────────┬───────────┬──────────┬─────┘
   │          │           │          │
   ▼          ▼           ▼          ▼
  H-1        E-1         M-1        U-1
 (허브)    (게임목록)   (내기록메인)   (프로필)
```

### 홈 허브 흐름

```
홈(H-1) — 모든 섹션의 요약을 보여주는 허브
  │
  ├── 🔍 빠른 검색바
  │     └── 입력 → 게임 목록(E-1) 검색 결과
  │
  ├── 🎲 인기 게임 섹션
  │     ├── 게임 카드 탭 (가격 포함) → 게임 상세(E-2)
  │     └── "전체 보기" → 게임 목록(E-1)
  │
  ├── 📚 내 최근 기록 섹션 (로그인 시)
  │     ├── 최근 플레이 로그 → 로그 상세(M-6)
  │     ├── "+ 기록하기" → 로그 작성(M-4)
  │     └── "전체 보기" → 내 기록 메인(M-1)
  │
  └── 🛒 추천 게임 / 가격 정보 섹션
        └── 게임 카드 탭 → 게임 상세(E-2) → 구매 링크
```

### 인증 흐름

```
앱 진입
  │
  ├── 비로그인 → 홈(H-1) 열람 가능 (읽기만)
  │                │
  │                ├── 기록/평점 등 시도
  │                │         │
  │                │         ▼
  │                │     로그인(A-1)
  │                │         │
  │                │    ┌────┴────┐
  │                │    ▼         ▼
  │                │  이메일    소셜로그인
  │                │  로그인    (카카오/구글)
  │                │    │         │
  │                │    ▼         │
  │                │  비번찾기    │
  │                │  (A-3→A-4)  │
  │                │    │         │
  │                │    ▼         ▼
  │                └──→ 홈(H-1) ←─┘
  │
  └── 회원가입(A-2) → 홈(H-1)
```

### 탐색 흐름

```
게임 목록(E-1)
  │
  ├── 검색바 입력 → 결과 필터링 (같은 화면)
  ├── 필터 적용 → 결과 갱신 (같은 화면)
  │
  └── 게임 카드 탭
        │
        ▼
      게임 상세(E-2)
        │
        ├── 구매하기 → 외부 쇼핑몰 링크 (새 탭)
        ├── 별점 등록 → 평점 모달(E-3) → 저장 → E-2
        ├── 컬렉션 추가 → "보유/위시" 선택 바텀시트 → 저장
        ├── 북마크 → 토글
        └── 플레이 로그 작성 → M-4로 이동 (게임 프리필)
```

### 내 기록 흐름

```
내 기록 메인(M-1)
  │
  ├── 컬렉션 요약 → 내 컬렉션(M-2)
  │                    │
  │                    ├── [보유] 탭
  │                    ├── [위시리스트] 탭
  │                    └── [플레이완료] 탭
  │                         └── 게임 탭 → 게임 상세(E-2)
  │
  ├── 플레이 로그 → 로그 목록(M-3)
  │                    │
  │                    ├── 로그 탭 → 로그 상세(M-6)
  │                    │               │
  │                    │               ├── 수정 → M-5 → 저장 → M-6
  │                    │               └── 삭제 → 확인 모달 → M-3
  │                    │
  │                    └── + 기록하기 → 로그 작성(M-4)
  │                                       │
  │                                       ├── 게임 선택 (검색 모달)
  │                                       ├── 날짜 선택
  │                                       ├── 함께한 사람 입력
  │                                       ├── 장소 입력
  │                                       ├── 별점 선택
  │                                       ├── 메모 입력
  │                                       └── 저장 → M-3
  │
  ├── 내 평점 → 평점 목록(M-7)
  │               └── 게임 탭 → 게임 상세(E-2)
  │
  └── 내 북마크 → 북마크 목록(M-8)
                    └── 게임 탭 → E-2
```

### 계정 흐름

```
프로필(U-1)
  │
  ├── 프로필 수정 → U-2 → 저장 → U-1
  │                  │
  │                  ├── 닉네임 변경
  │                  ├── 프로필 이미지 변경 (업로드/삭제)
  │                  └── 자기소개 변경
  │
  ├── 알림 🔔 → 알림 목록(U-3)
  │               └── 알림 탭 → 해당 로그(M-6) 또는 게임(E-2)
  │
  └── 설정 ⚙️ → 설정 허브(U-4)
                  │
                  ├── 계정 정보 변경(U-5)
                  │     ├── 이메일 변경
                  │     └── 소셜 계정 연결/해제
                  │
                  ├── 비밀번호 변경(U-6)
                  │     └── 현재 비번 → 새 비번 → 저장
                  │
                  ├── 알림 설정(U-7)
                  │     └── 알림 종류별 on/off
                  │
                  ├── 회원 탈퇴(U-8)
                  │     └── 사유 선택 → 비번 확인 → 탈퇴 완료
                  │
                  ├── 이용약관(S-1)
                  ├── 개인정보처리방침(S-2)
                  ├── 서비스 가이드라인(S-3)
                  └── 로그아웃 → 확인 모달 → A-1
```

### 공통 모달 / 바텀시트 목록

```
전체 화면에서 호출 가능:
  │
  ├── 게임 검색 모달        ← 플레이 로그에서 게임 선택 시
  ├── 평점 등록 바텀시트     ← 게임 상세에서 별점 등록 시
  ├── 컬렉션 추가 바텀시트   ← 게임 상세에서 "보유/위시" 선택 시
  ├── 삭제 확인 모달         ← 로그 삭제 시
  ├── 로그아웃 확인 모달     ← 설정에서 로그아웃 시
  └── 토스트 메시지          ← 저장 완료, 에러 등
```
