### 필터 칩

```
  [인원수 ▾]  [가격대 ▾]  [필터 더보기 ▾]
```

**기본 칩 (항상 노출):**

| 칩 | 옵션 |
|-----|------|
| 인원수 | 1인 / 2인 / 3~4인 / 5인 이상 |
| 가격대 | ~2만원 / 2~4만원 / 4~6만원 / 6만원~ |

**칩 스타일:**

| 상태 | 스타일 |
|------|--------|
| 기본 | gray-100 배경, gray-700 텍스트, radius 9999px |
| 선택됨 | primary-100 배경, primary-600 텍스트, primary-200 보더 |
| 높이 | 32px, 패딩 8px 14px |

**칩 탭 시:** 드롭다운 (데스크톱) 또는 바텀시트 (모바일)로 옵션 표시

## 3. 각 요소 상세

### 하단 구매바 (모바일)

```
┌─────────────────────────────────────┐
│  35,000원              [ 구매하기 → ] │
└─────────────────────────────────────┘
```

| 요소 | 규격 |
|------|------|
| 위치 | 하단 고정, 탭바 바로 위 |
| 높이 | 64px |
| 배경 | white |
| 상단 보더 | gray-200 (1px) |
| 가격 | h2 (18px Bold), gray-900 |
| 구매 버튼 | Primary (md), "구매하기 →" |
| 좌우 패딩 | 16px |

**동작:**
- 구매하기 탭 → 외부 쇼핑몰 새 탭으로 열림
- 구매 링크 없을 시 → 버튼 비활성 + "판매처 준비 중"

### 구매 사이드 카드 (데스크톱)

```

│  │  🎲 카탄                     │   │  ← 게임명 표시
│  └─────────────────────────────┘   │
```

### 데스크톱 (중앙 모달)

```
┌───────────────────────────────┐
│  내 평점 등록              ✕   │
│                               │
│  🎲 카탄                      │
│                               │
│     ☆   ☆   ☆   ☆   ☆      │
│                               │
│     탭하여 별점을 선택하세요     │
│                               │
│  [ 저장하기 ]                  │
│  [삭제하기]                    │
└───────────────────────────────┘

너비: 400px, 중앙 정렬
```

| 요소 | 규격 |
|------|------|
| 별 크기 | 40px, 간격 12px |
| 별 색상 | 빈 별 gray-200, 채워진 별 accent-400 |
| 호버 | 마우스 올린 별까지 채워짐 (프리뷰) |
| 탭 | 탭한 별까지 확정, 스케일 애니메이션 (1.1→1.0) |
| 저장 버튼 | 별 선택 전 비활성 |

## 5. 검색 동작

| 단계 | 설명 |
|------|------|
| 입력 시작 | 300ms 디바운스 후 검색 실행 |
| 결과 표시 | 같은 화면에서 그리드 갱신 |
| 결과 없음 | 빈 상태 표시 (아래 참고) |
| 검색어 삭제 | X 버튼 또는 전체 삭제 → 기본 목록으로 복귀 |

## 6. 모바일 ↔ 데스크톱 차이 정리

| 요소 | 모바일 | 데스크톱 |
|------|--------|---------|
| 게임 목록 그리드 | 2열 | 4열 |
| 필터 | 칩 2개 + "더보기" 바텀시트 | 칩 5개 한 줄 + 드롭다운 |
| 게임 이미지 | 전체 너비 (1:1) | 좌측 400×400 |
| 게임 정보 | 이미지 아래 세로 나열 | 이미지 우측 가로 배치 |
| 구매 버튼 | 하단 고정 바 | 우측 sticky 카드 |
| 평점 모달 | 바텀시트 | 중앙 모달 |
| 연관 게임 | 가로 스크롤 | 4열 그리드 |
