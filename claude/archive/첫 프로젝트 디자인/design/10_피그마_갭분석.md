## 1. 화면 전체 누락

와이어프레임에 있으나 피그마에 아예 없는 화면.

| ID | 화면 | 경로 | 우선순위 |
|---|---|---|---|
| **A-4** | 비밀번호 재설정 | `/reset-password` | 🔴 High |
| **U-3** | 알림 | `/notifications` | 🔴 High |
| **U-4** | 설정 허브 | `/settings` | 🔴 High |
| **U-7** | 알림 설정 | `/settings/notifications` | 🟡 Medium |

**A-4 비밀번호 재설정** — 비밀번호 찾기(A-3) 프레임에 이메일 전송 완료 상태는 있지만, 링크 클릭 후 진입하는 "새 비밀번호 입력" 화면이 없음. 별도 데스크톱+모바일 필요.

**U-4 설정 허브** — 설정 메뉴 목록 화면 자체가 없음. 계정 정보 변경/비밀번호 변경은 있지만 이들로 진입하는 허브가 없음.

## 3. GNB 구조 불일치 (심각)

피그마 페이지마다 GNB 레이블이 제각각. 와이어프레임 기준(`04_와이어프레임_공통.md`)에 맞게 통일 필요.

| 프레임 | 현재 GNB 레이블 | 기준 GNB 레이블 |
|---|---|---|
| 홈, 탐색 (1:370, 1:670) | 홈, 탐색, 내기록 | **기준에 부합** |
| 게임 상세 v2 (1:1590) | Discover, Library, Collections, Community | ❌ 영문, 다른 구조 |
| 회원 탈퇴 (1:4125) | Collection, Sessions, Market, Library | ❌ 또 다른 구조 |
| 비밀번호 찾기 (1:3424) | Explore, Collection, Sign In | ❌ 비로그인 상태, 메뉴 다름 |

**와이어프레임 기준 GNB:**
- 로그인 상태: `BoardLog 로고 | 홈 탐색 내기록 | 🔔 👤닉네임▾`
- 비로그인 상태: `BoardLog 로고 | 홈 탐색 | [로그인] [회원가입]`

## 4. 요소 수준 누락

존재하는 화면 안에서 와이어프레임 정의 대비 빠진 요소들.

### 4.1 게임 상세 (E-2)

| 누락 요소 | 와이어프레임 정의 |
|---|---|
| **내 평점 등록하기 버튼** | Secondary 버튼 → E-3 모달 트리거 |
| **컬렉션 추가 바텀시트** | "보유중 / 위시리스트 / 플레이완료" 선택 |
| **평점 등록/수정 모달 (E-3)** | 별 5개 선택, 저장/삭제 버튼 포함 |

현재 피그마 게임 상세(1:1590)에는 "컬렉션에 추가" + "플레이 로그 작성" 버튼만 있고 평점 등록 버튼이 없음.

### 4.2 홈 (H-1)

| 누락 요소 | 와이어프레임 정의 |
|---|---|
| **검색바** | 홈 최상단 또는 배너 아래 검색 인풋 |
| **비로그인 로그인 유도 카드** | 내 최근 기록 섹션 대신 표시 |

### 4.3 내 기록 (M-1~M-8)

| 누락 요소 | 와이어프레임 정의 |
|---|---|
| **각 탭별 빈 상태(empty state)** | 컬렉션 빔 / 로그 빔 / 평점 빔 / 북마크 빔 4종 |
| **M-5 플레이 로그 수정 화면** | 작성 폼과 동일하나 헤더 "기록 수정", 기존값 프리필 |

### 4.4 공통 UI 상태

| 누락 요소 | 와이어프레임 정의 |
|---|---|
| **비로그인 GNB 상태** | `[로그인] [회원가입]` 버튼 있는 GNB |
| **비로그인 내기록/프로필 진입 시 유도 화면** | 🔒 인라인 로그인 유도 카드 |
| **토스트 메시지** | 성공(초록)/에러(빨강) 2종, 모바일+데스크톱 위치 다름 |
| **로그아웃 확인 모달** | "로그아웃 하시겠어요?" + 취소/확인 |
| **알림 뱃지** | GNB 알림 아이콘 우상단 빨간 점 |
| **스켈레톤 UI** | 게임 카드, 리스트 아이템 로딩 상태 |

## 5. 피그마 버전 중복 정리 필요

게임 상세 화면이 2가지 버전으로 중복되어 있음.

| 버전 | 프레임 ID | GNB | 특이점 |
|---|---|---|---|
| v1 | 1:670 (탐색+게임상세) | 한국어 GNB | 탐색 목록 + 상세 합쳐진 롱페이지 |
| v2 | 1:1590, 1:1978 | 영문 GNB | 게임 상세만, 더 완성도 높음 |

**v1 (1:670)을 참고용으로만 두고, v2 기준으로 GNB/요소 수정 후 확정하는 것을 권장.**

## 6. 작업 우선순위 요약

### 🔴 즉시 필요 (개발 전 필수)

1. **GNB 전 페이지 통일** — 영문/국문 혼재, 메뉴 구조 불일치 수정
2. **U-4 설정 허브** — 데스크톱 + 모바일 신규 제작
3. **U-3 알림** — 데스크톱 + 모바일 신규 제작
4. **A-4 비밀번호 재설정** — 데스크톱 + 모바일 신규 제작
5. **E-2 평점 등록 버튼 + E-3 모달** — 게임 상세에 추가
6. **E-1 게임 목록 모바일** — 신규 제작

### 🟡 개발 시작 전까지 필요

7. **공통 빈 상태(empty state)** 4종 — 내 기록 탭별
8. **비로그인 상태** UI — GNB, 내기록 유도 화면
9. **반응형 누락** — A-3/U-5/U-6/U-8 모바일 버전
10. **토스트 메시지** — 성공/에러 2종

### 🟢 여유 있을 때

11. **M-5 플레이 로그 수정** — 작성 화면에 수정 상태 추가
12. **알림 뱃지** — 탭바/GNB 아이콘 상태
13. **스켈레톤 UI** — 로딩 상태
14. **U-7 알림 설정** — 모바일 버전 추가
15. **피그마 중복 프레임 정리** — 게임 상세 v1/v2 정리
