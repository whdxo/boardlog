## 2. 각 섹션 상세

### 2.1 검색바

```
┌─────────────────────────────────────┐
│  🔍 보드게임을 검색해보세요              │
└─────────────────────────────────────┘
```

| 요소 | 규격 |
|------|------|
| 높이 | 44px |
| 배경 | gray-50 |
| 보더 | gray-200 (1px), 포커스 시 primary-500 |
| radius | 8px |
| 아이콘 | Search, 16px, gray-400 |
| 플레이스홀더 | "보드게임을 검색해보세요", 14px, gray-400 |
| 좌우 마진 | 16px |

**동작:**
- 탭하면 게임 목록(E-1) 페이지로 이동, 검색바에 자동 포커스
- 홈에서는 입력 불가 (이동 트리거 역할만)

---

### 2.2 배너 슬라이드

```
┌─────────────────────────────────────┐
│                                     │
│  [배너 이미지]                        │
│  신작 보드게임 "위잉" 출시!             │
│  전략과 블러핑의 만남                   │
│                                     │
│  [ 자세히 보기 ]                      │
│                                     │
│              ● ○ ○                  │
└─────────────────────────────────────┘
```

| 요소 | 규격 |
|------|------|
| 비율 | 16:9 (모바일) / 3:1 (데스크톱) |
| radius | 12px |
| 좌우 마진 | 16px (모바일) / 0 (데스크톱, 풀와이드) |
| 인디케이터 | 하단 중앙, 점(dot) 형태, 활성 primary-500 / 비활성 gray-300 |
| 자동 전환 | 5초 간격, 스와이프로 수동 전환 가능 |
| 최대 개수 | 5장 |
| 배너 내 텍스트 | 이미지 위 오버레이 (하단 그라데이션) |
| CTA 버튼 | 있으면 "자세히 보기" → 게임 상세(E-2) 또는 외부 링크 |

**배너 유형:**
- 신작 게임 소개 → 게임 상세(E-2)
- 할인/프로모션 → 외부 쇼핑몰 (새 탭)
- 이벤트/공지 → 정적 페이지 or 외부

### 2.3 인기 게임

```
  🔥 인기 게임                    전체 보기 >
  ┌──────────┐ ┌──────────┐ ┌──────────┐
  │          │ │          │ │          │
  │  [썸네일]  │ │  [썸네일]  │ │  [썸네일]  │
  │          │ │          │ │          │
  ├──────────┤ ├──────────┤ ├──────────┤
  │ 카탄      │ │ 스플렌더  │ │ 윙스팬    │
  │ ★ 4.2    │ │ ★ 3.8    │ │ ★ 4.5    │
  │ 32,000원  │ │ 28,000원  │ │ 45,000원  │
  └──────────┘ └──────────┘ └──────────┘
       ←  가로 스크롤  →
```

| 요소 | 규격 |
|------|------|
| 섹션 제목 | h1 (22px Bold), 좌측 |
| "전체 보기" | caption (12px), gray-600, 우측, 탭 → E-1 |
| 카드 너비 | 140px (모바일) / 200px (데스크톱) |
| 카드 간격 | 12px |
| 스크롤 | 가로 스크롤, 스냅, 스크롤바 숨김 |
| 좌측 패딩 | 16px (첫 카드 시작점) |
| 카드 구성 | 썸네일(3:4) + 게임명(h3) + 평점(caption) + 가격(body Bold) |
| 가격 색상 | primary-500 |
| 표시 개수 | 최대 10개 |

**정렬 기준:** 평점순 (초기에는 BGG 평점, 이후 자체 평점)

**카드 탭 시:** 게임 상세(E-2)로 이동

### 2.4 내 최근 기록 (로그인 시)

```
  📚 내 최근 기록                  전체 보기 >
  ┌─────────────────────────────────┐
  │ 🎲 카탄        2026.03.28       │
  │ 👥 민수, 지영  📍 보드게임 카페    │
  ├─────────────────────────────────┤
  │ 🎲 스플렌더     2026.03.25       │
  │ 👥 지영        📍 집             │
  └─────────────────────────────────┘
  
  [ + 새 기록 작성하기 ]
```

| 요소 | 규격 |
|------|------|
| 섹션 제목 | h1 (22px Bold) |
| "전체 보기" | caption (12px), gray-600, 탭 → M-1 |
| 로그 아이템 높이 | 60px |
| 게임명 | h3 (16px SemiBold) |
| 날짜 | caption (12px), gray-400, 우측 정렬 |
| 함께한 사람 | caption (12px), gray-600 |
| 장소 | caption (12px), gray-600, 📍 아이콘 |
| 구분선 | gray-100 (1px) |
| 표시 개수 | 최대 3개 |
| 기록 버튼 | Secondary 버튼 (md), 전체 너비 |

**비로그인 시:** 이 섹션 대신 로그인 유도 카드 표시

```
  ┌─────────────────────────────────┐
```

## 5. 홈 데이터 로딩

| 섹션 | 로딩 방식 |
|------|----------|
| 검색바 | 정적 (로딩 없음) |
| 배너 | SSR (서버에서 미리 렌더) |
| 인기 게임 | SSR + 캐싱 (1시간) |
| 내 최근 기록 | CSR (로그인 유저만, 클라이언트 요청) |
| 추천 게임 | SSR + 캐싱 (1시간) |

각 섹션은 독립적으로 로딩되며, 스켈레톤 UI로 대체 표시.
