# 차별화 전략

## 보드라이프 vs BoardLog

| 구분 | 보드라이프 | BoardLog |
|-----|-----------|---------|
| 구조 | 게시판형 (올드) | 피드형 (모바일 중심) |
| 핵심 | 커뮤니티 + DB | 기록 + 커뮤니티 + 펀딩 |
| 펀딩 | 없음 | 핵심 기능 |
| 추천 | 없음 | 취향 기반 추천 (Phase 2) |
| UX | 복잡, 분산 | 단순, 집중 |
| 데이터 | 유저 참여형 (정확도 낮음) | 큐레이션 + BGG 연동 |

---

## 핵심 차별화 포인트 3가지

### 1. 펀딩 연결
보드라이프에 없는 기능. 보드게임 제작자와 팬을 직접 연결하는 구조.
- 텀블벅/킥스타터처럼 독립 플랫폼이 아니라 "커뮤니티 안에 펀딩"
- 유저가 응원하고 싶은 게임에 바로 후원 가능
- 제작자는 피드백을 커뮤니티에서 바로 받을 수 있음

### 2. 내 기록 중심 설계
단순 컬렉션 관리가 아니라 "보드게임 다이어리"
- 언제, 누구와, 몇 번 플레이했는지
- 플레이 메모 및 감상 기록
- 이게 쌓이면 유저가 못 떠남 (락인)

### 3. 깔끔한 UX (모바일 우선)
보드라이프의 가장 큰 약점은 UX.
- 카드형 게임 탐색
- 인스타그램식 피드 커뮤니티
- 복잡한 메뉴 없이 탭 5개로 모든 것

---

## 타겟 유저

### 메인 타겟
- 20~40대 보드게임 취미 유저
- 게임 기록을 남기고 싶은 사람
- 새로운 게임을 발견하고 싶은 사람

### 세컨더리 타겟
- 보드게임 제작자 (펀딩 기능으로 유입)
- 보드게임 카페 운영자 (Phase 2+)

## 경쟁 서비스 분석

| 서비스 | 강점 | 약점 | 우리 포지션 |
|-------|------|------|------------|
| 보드라이프 | 트래픽, 데이터 | UX, 기능 분산 | UX + 펀딩으로 차별화 |
| BoardGameGeek | 글로벌 DB, 데이터 | 한국어 없음, UX 최악 | 국내 특화 |
| 텀블벅 | 펀딩 전문 | 보드게임 특화 없음 | 커뮤니티 + 펀딩 결합 |
| 당근마켓 | 중고거래 | 보드게임 특화 없음 | Phase 2에서 틈새 공략 |
