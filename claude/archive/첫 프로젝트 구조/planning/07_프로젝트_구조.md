## Route Groups 설명

Next.js App Router의 Route Group `(폴더명)` 사용. URL에는 영향 없고 레이아웃만 다르게 적용.

| 그룹 | 레이아웃 | 포함 페이지 |
|------|---------|------------|
| `(auth)` | 로고만 있는 심플 레이아웃 (GNB/탭바 없음) | 로그인, 회원가입, 비번찾기 |
| `(main)` | GNB(데스크톱) + 탭바(모바일) + 푸터 | 게임, 내기록, 프로필, 설정, 알림 |
| `(static)` | 심플 헤더 + 본문 중앙 (max 768px) | 약관, 개인정보, 가이드라인 |

홈(`/`)은 루트 `page.tsx`에 있고 `(main)` 레이아웃을 공유.

## 컴포넌트 분류

### `components/ui/` — 기본 UI 컴포넌트
shadcn/ui 기반 + 커스텀. 디자인 시스템 문서 기준.

```
Button.tsx        — Primary, Secondary, Ghost, Danger
Input.tsx         — 텍스트, 비밀번호(👁), 에러 상태
Modal.tsx         — 데스크톱 중앙 모달
BottomSheet.tsx   — 모바일 바텀시트
Dropdown.tsx      — 드롭다운 메뉴
Toggle.tsx        — on/off 토글 스위치
Checkbox.tsx      — 체크박스
```

### `components/layout/` — 레이아웃
```
GNB.tsx           — 데스크톱 상단 네비게이션
TabBar.tsx        — 모바일 하단 탭바
Header.tsx        — 모바일 상단 헤더 (뒤로가기 등)
Footer.tsx        — 데스크톱 푸터
MainLayout.tsx    — (main) 그룹 레이아웃 래퍼
AuthLayout.tsx    — (auth) 그룹 레이아웃 래퍼
```

### `components/game/` — 게임 관련
```
GameCard.tsx      — 게임 카드 (목록, 홈에서 재사용)
GameGrid.tsx      — 게임 카드 그리드 (2열/4열)
GameSearchModal.tsx — 게임 검색 모달 (로그 작성에서 사용)
StarRating.tsx    — 별점 (읽기전용 + 입력용)
RatingModal.tsx   — 평점 등록 모달
CollectionSheet.tsx — 컬렉션 추가 바텀시트
FilterChips.tsx   — 필터 칩 (인원수, 가격대)
FilterSheet.tsx   — 필터 더보기 바텀시트
PurchaseBar.tsx   — 하단 고정 구매바 (모바일)
PurchaseCard.tsx  — 구매 사이드 카드 (데스크톱)
```

### `components/log/` — 플레이 로그 관련
```
LogCard.tsx       — 로그 카드 (목록에서 사용)
LogForm.tsx       — 로그 작성/수정 폼
PlayerChips.tsx   — 함께한 사람 칩 입력
```

### `components/common/` — 공통
```
Toast.tsx         — 토스트 메시지
EmptyState.tsx    — 빈 상태 (아이콘+메시지+CTA)
Skeleton.tsx      — 스켈레톤 로딩
SearchBar.tsx     — 검색바 (홈, 게임목록 공용)
BannerSlide.tsx   — 배너 슬라이드 (홈)
LoginPrompt.tsx   — 로그인 유도 화면
ScrollToTop.tsx   — 맨 위로 FAB
```

## 주요 디렉토리 역할

| 디렉토리 | 역할 | 예시 |
|---------|------|------|
| `lib/` | API 호출, 유틸 함수 | `api.ts`, `utils.ts`, `format.ts` |
| `hooks/` | 커스텀 React 훅 | `useAuth.ts`, `useToast.ts`, `useInfiniteScroll.ts` |
| `types/` | TypeScript 인터페이스 | `game.ts`, `user.ts`, `log.ts` |
| `constants/` | 상수값 | `routes.ts`, `filters.ts`, `colors.ts` |
| `styles/` | 추가 CSS/폰트 | `pretendard.css` |
