# BoardLog — 피그마 → 코드 변환 계획

## Context

피그마에 완성된 BoardLog UI 디자인을 Next.js 16.2.2 + React 19 + Tailwind v4 + TypeScript 코드로 변환한다.
현재 프로젝트는 create-next-app 기본 상태(컴포넌트 없음, UI 라이브러리 미설치)이며,
docs/design/ 에 디자인 시스템·와이어프레임·갭 분석이 완비되어 있다.
피그마 없는 화면(A-4, U-3, U-4, U-7 등)은 와이어프레임 스펙으로 구현한다.

---

## 주요 기술 결정 사항

- **Tailwind v4**: `tailwind.config.js` 없음 → `globals.css`의 `@theme` 블록에서 토큰 정의
- **Next.js 16 Breaking**: `params` / `searchParams` 모두 `Promise` → 페이지 컴포넌트에서 반드시 `await`
- **Turbopack 기본값**: `next dev` 가 Turbopack으로 실행됨 (별도 플래그 불필요)
- **shadcn/ui**: Tailwind v4 호환 버전으로 설치 (`npx shadcn@latest init`)
- **Pretendard 폰트**: `next/font/local` 로 로드 (Google Fonts 미지원 한국어 폰트)
- **라우트 그룹**: `(auth)` / `(main)` / `(static)` 으로 레이아웃 분리
