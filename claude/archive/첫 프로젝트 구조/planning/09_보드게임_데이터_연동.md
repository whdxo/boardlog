# 보드게임 데이터 연동

> BGG API를 활용한 보드게임 데이터 수집 및 우리 DB 매핑 전략

---

## 1. 데이터 소스 요약

| 소스 | 용도 | 형식 | 비용 |
|------|------|------|------|
| **BGG XML API2** | 메인 데이터 소스 (검색, 상세, 인기) | XML | 무료 |
| **Kaggle 데이터셋** | 초기 DB 시딩 (2만+ 게임 일괄) | CSV | 무료 |
| **bgg-xml-api-client** (npm) | BGG API JS 래퍼 (XML→JSON 자동 변환) | - | 무료 |
| **직접 입력** | BGG에 없는 국내 게임 | - | - |

## 2. BGG API 엔드포인트

Base URL: `https://boardgamegeek.com/xmlapi2/`

### 2.1 게임 상세 — `/thing`

가장 많이 쓸 엔드포인트. 게임의 모든 정보를 제공.

```
GET /xmlapi2/thing?id=174430&stats=1&type=boardgame
```

**파라미터:**

| 파라미터 | 설명 | 예시 |
|----------|------|------|
| `id` | BGG 게임 ID (쉼표로 최대 20개) | `id=174430,13,822` |
| `type` | `boardgame` | |
| `stats` | `1`이면 평점/랭킹 포함 | 필수로 넣기 |

**반환 데이터 → 우리 DB 매핑:**

| BGG 필드 | 우리 DB 필드 | 예시 값 |
|----------|-------------|---------|
| `name[@type='primary']` | `title` (영문) | Catan |
| `name[@type='alternate']` | `title_kr` (한글 있으면) | 카탄 |
| `description` | `description` | 게임 설명 (HTML 디코딩 필요) |
| `image` | `image_url` | https://cf.geekdo-images.com/... |
| `thumbnail` | 썸네일용 | https://cf.geekdo-images.com/... |
| `minplayers` | `min_players` | 2 |
| `maxplayers` | `max_players` | 4 |
| `minplaytime` | `play_time_min` | 60 |
| `maxplaytime` | `play_time_max` | 120 |
| `minage` | `min_age` | 10 |
| `yearpublished` | `year_published` | 1995 |
| `item@id` | `bgg_id` | 13 |
| `statistics/ratings/average` | `avg_rating` | 7.15 (10점→5점 변환 필요) |
| `statistics/ratings/usersrated` | `rating_count` | 98234 |
| `statistics/ratings/averageweight` | (참고용) 난이도 | 2.32 |
| `statistics/ratings/ranks/rank[@name='boardgame']` | (참고용) BGG 랭킹 | 423 |
| `link[@type='boardgamedesigner']` | `designer` | Klaus Teuber |
| `link[@type='boardgamepublisher']` | `publisher` | Kosmos |
| `link[@type='boardgamecategory']` | `tags` (genre) | Negotiation, Economic |
| `link[@type='boardgamemechanic']` | `tags` (mechanic) | Dice Rolling, Trading |

**추가 유용한 데이터 (Phase 2+):**

| BGG 필드 | 용도 |
|----------|------|
| `statistics/ratings/owned` | "N명이 보유" 표시 |
| `statistics/ratings/wishing` | "N명이 위시리스트에" 표시 |
| `poll[name='suggested_numplayers']` | "3~4인 추천" 뱃지 |
| `poll[name='language_dependence']` | 한국어판 필요 여부 |
| `link[@type='boardgameexpansion']` | 확장팩 연결 |
| `link[@type='boardgamefamily']` | 시리즈 게임 그룹 |

### 2.2 검색 — `/search`

유저가 게임 이름으로 검색할 때 사용.

```
GET /xmlapi2/search?query=catan&type=boardgame
```

**반환:** ID + 이름 + 출시년도만. 상세 정보는 `/thing`으로 재호출 필요.

| 필드 | 설명 |
|------|------|
| `item@id` | BGG ID |
| `item/name@value` | 게임 이름 |
| `item/yearpublished@value` | 출시년도 |

**우리 사용 방식:**
```
유저 검색 → /search로 ID 목록 획득 → 우리 DB에 있으면 바로 표시
                                     → 없으면 /thing으로 상세 수집 후 DB 저장
```

---

### 2.3 인기 게임 — `/hot`

홈 "인기 게임" 섹션용. 현재 BGG에서 뜨는 게임 50개.

```
GET /xmlapi2/hot?type=boardgame
```

**반환:** 50개 고정, 순위 + 이름 + 썸네일 + 출시년도

| 필드 | 설명 |
|------|------|
| `item@rank` | 핫 순위 (1~50) |
| `item@id` | BGG ID |
| `item/name@value` | 게임 이름 |
| `item/thumbnail@value` | 썸네일 |
| `item/yearpublished@value` | 출시년도 |

**우리 사용 방식:** 하루 1회 수집 → DB 캐싱 → 홈에서 표시

## 3. API 제한 및 규칙

| 항목 | 내용 |
|------|------|
| API 키 | 불필요 |
| Rate Limit | 공식 명시 없으나 **5초에 1회** 권장 |
| 배치 요청 | `/thing?id=1,2,3,...` 최대 20개 |
| 202 응답 | collection 등에서 발생 → 5~10초 후 재시도 |
| 응답 형식 | XML (JSON 없음) |
| User-Agent | 커스텀 설정 권장 (`BoardLog/1.0`) |
| description 인코딩 | HTML 엔티티 (`&amp;` 등) 디코딩 필요 |

## 4. 평점 변환

BGG는 **10점 만점**, 우리는 **5점 만점**.

```
변환 공식: 우리 평점 = BGG 평점 / 2

예시:
  BGG 7.15 → BoardLog 3.6
  BGG 8.40 → BoardLog 4.2
  BGG 6.00 → BoardLog 3.0

소수점: 첫째 자리까지 (반올림)
```

> 우리 유저가 남긴 자체 평점과 BGG 평점은 별도 관리.
> 자체 평점이 충분히 쌓이면 자체 평점 우선 표시.

## 5. 데이터 수집 전략

### Phase 1 — 초기 시딩 (서비스 런칭 전)

```
1. Kaggle 데이터셋 다운로드 (2만+ 게임 CSV)
   ↓
2. CSV → DB 일괄 insert (Prisma seed 스크립트)
   - 게임명, 인원, 시간, 연령, 평점, 이미지 URL, BGG ID
   ↓
3. BGG /hot 호출 → 인기 50개 게임 상세 보강
   ↓
4. 한국어 게임명 매핑
   - BGG alternate name에서 한국어 추출
   - 없으면 수동 입력 (인기 게임 우선)
```

### Phase 2 — 운영 중 자동 수집

```
유저가 검색 시:
  1. 우리 DB에서 먼저 검색
  2. 결과 부족 → BGG /search 호출
  3. BGG 결과 중 우리 DB에 없는 게임 → /thing으로 상세 수집
  4. DB에 저장 (자동 확장)

일일 배치 (cron):
  - /hot 50개 갱신 (하루 1회)
  - 기존 DB 게임 평점 업데이트 (주 1회, 배치)
```

### Phase 3 — 한국 게임 데이터

```
- BGG에 없는 국내 전용 게임은 관리자가 직접 입력
- 국내 퍼블리셔 (코리아보드게임즈, 다이브다이스 등) 게임 우선 등록
- 가격 정보: 네이버 쇼핑 링크 수동 등록 (Phase 2에서 자동화 검토)
```

## 7. npm 패키지 활용

### bgg-xml-api-client

```bash
npm install bgg-xml-api-client
```

```typescript
import { BggClient } from 'bgg-xml-api-client';

const bgg = BggClient.create();

// 게임 상세
const game = await bgg.thing({ id: 13, stats: 1, type: 'boardgame' });

// 검색
const results = await bgg.search({ query: 'catan', type: 'boardgame' });

// 핫 리스트
const hot = await bgg.hot({ type: 'boardgame' });
```

**장점:**
- XML → JSON 자동 변환
- 202 응답 자동 재시도
- TypeScript 지원

## 8. Kaggle 초기 시딩 데이터

### 데이터셋

Kaggle에서 "BoardGameGeek" 검색 → 인기 데이터셋 다운로드

**주요 컬럼 (CSV):**

| 컬럼 | 매핑 |
|------|------|
| `id` | `bgg_id` |
| `name` | `title` |
| `year_published` | `year_published` |
| `min_players` | `min_players` |
| `max_players` | `max_players` |
| `avg_time` / `playing_time` | `play_time_min` |
| `min_age` | `min_age` |
| `description` | `description` |
| `thumbnail` | `image_url` |
| `average_rating` | `avg_rating` (10→5점 변환) |
| `users_rated` | `rating_count` |
| `categories` | `tags` (genre) |
| `mechanics` | `tags` (mechanic) |
| `designers` | `designer` |
| `publishers` | `publisher` |
| `board_game_rank` | 정렬 기준 |

### 시딩 스크립트 (예시)

```typescript
// prisma/seed.ts
import { parse } from 'csv-parse/sync';
import { readFileSync } from 'fs';
import { prisma } from '../src/lib/prisma';

const csv = readFileSync('data/bgg_games.csv', 'utf-8');
const games = parse(csv, { columns: true });

for (const game of games) {
  await prisma.boardGame.upsert({
    where: { bgg_id: parseInt(game.id) },
    update: {},
    create: {
      title: game.name,
      description: game.description || '',
      min_players: parseInt(game.min_players) || 0,
      max_players: parseInt(game.max_players) || 0,
      play_time_min: parseInt(game.avg_time) || 0,
      play_time_max: parseInt(game.max_time) || 0,
      min_age: parseInt(game.min_age) || 0,
      publisher: game.publishers?.split(',')[0] || '',
      designer: game.designers?.split(',')[0] || '',
      year_published: parseInt(game.year_published) || 0,
      image_url: game.thumbnail || '',
      bgg_id: parseInt(game.id),
      avg_rating: parseFloat(game.average_rating) / 2 || 0,
      rating_count: parseInt(game.users_rated) || 0,
    },
  });
}
```

## 9. 데이터 갱신 주기

| 데이터 | 갱신 주기 | 방법 |
|--------|----------|------|
| 핫 리스트 (인기 50) | 하루 1회 | cron → BGG `/hot` |
| 게임 평점/통계 | 주 1회 | cron → BGG `/thing` (배치) |
| 새 게임 추가 | 유저 검색 시 | 실시간 → BGG `/search` + `/thing` |
| 한국어 이름 | 수동 | 관리자 입력 |
| 가격 정보 | 수동 | 관리자 입력 (Phase 2 자동화) |
| 이미지 | 최초 수집 시 | BGG URL 사용 (자체 저장은 Phase 2) |
