# 정보 구조 (IA) & 메뉴 설계

## 최상위 네비게이션 (MVP 기준)

```
홈
├── 탐색 (보드게임)
├── 커뮤니티
├── 펀딩
└── 내 기록
```

---

## 각 섹션 상세

### 🏠 홈
- 추천 게임 (취향 기반 or 인기순)
- 최근 커뮤니티 핫 게시글
- 진행 중인 펀딩 프로젝트
- 최근 활동 피드 (팔로우 기반)

---

### 🎲 탐색 (보드게임 DB)
- 게임 목록
  - 정렬: 인기순 / 최신 / 평점 / 플레이 횟수
  - 필터: 인원수 / 플레이 시간 / 장르 / 연령
- 게임 상세 페이지
  - 기본 정보 (설명, 인원, 시간, 연령, 제작사)
  - 평점 & 리뷰
  - 플레이 통계 (등록한 유저 수, 평균 플레이 횟수)
  - 연관 게임 추천
- 검색
  - 키워드 검색
  - 태그 검색

---

### 💬 커뮤니티
- 피드 (전체 / 팔로우)
- 게시글 유형 (태그 기반)
  - 후기
  - 질문
  - 추천
  - 정보/소식
  - 자유
- 게시글 상세
  - 댓글
  - 좋아요 / 북마크
  - 연관 게임 태그
- 인기 게시글 (베스트)

---

### 🚀 펀딩
- 프로젝트 목록
  - 진행 중 / 예정 / 종료
  - 정렬: 마감 임박 / 달성률 / 최신
- 프로젝트 상세
  - 게임 소개
  - 제작자 정보
  - 펀딩 목표 & 달성률
  - 리워드 티어
  - 업데이트 & 댓글
- 프로젝트 등록 (제작자)

---

### 📚 내 기록 (My Log)
- 내 게임 컬렉션
  - 보유 / 위시리스트 / 플레이 완료
- 플레이 로그
  - 날짜, 같이 플레이한 사람, 결과, 메모
- 내 평점 & 리뷰 목록
- 활동 통계
  - 총 플레이 횟수 / 즐겨하는 장르 / 플레이 시간

---

## 사용자 계정
- 회원가입 / 로그인 (소셜 로그인 포함)
- 프로필
  - 내 정보
  - 팔로우 / 팔로워
  - 내 게시글 / 리뷰 / 로그
- 알림
- 설정
