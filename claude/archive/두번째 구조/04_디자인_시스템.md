## 5. 공통 컴포넌트

### 네비게이션

#### GNB (데스크톱)
- sticky, z-50, h-16
- 로고 | 메인 메뉴 | 검색 | 알림 | 프로필 드롭다운
- 메가 드롭다운 (2단 메뉴)

#### 모바일 하단 탭바 (5탭)
```
홈 | 탐색 | 커뮤니티 | 쇼핑 | 내 게임
```

#### 모바일 탑 헤더
- 로고 or 뒤로가기 버튼
- 페이지 타이틀
- 우측 슬롯 (검색/메뉴 등)

### 게임 카드
```
┌──────────────┐
│  3:4 이미지  │  — 커버
│   [뱃지]     │  — 컬렉션 상태 (있을 때)
│  [★ 4.5]    │  — 평점
└──────────────┘
  게임명 (2줄)
  2~4인 · 60분
  45,000원
```

#### 호버 효과 (데스크톱)
- `translateY(-4px) + shadow-lg`
- 빠른 컬렉션 추가 오버레이

### 버튼 변형
```
Primary   bg-primary-500 text-white
Outline   border-gray-200 text-gray-700
Ghost     text-gray-600 hover:bg-gray-100
Danger    bg-red-50 text-red-600
```

### 컬렉션 상태 선택 시트 (모바일 바텀시트)
8가지 상태를 아이콘+텍스트+설명으로 나열한 바텀시트

### 평점 모달
- 데스크톱: 중앙 다이얼로그
- 모바일: 바텀시트
- 별점 (0.5 단위, 10점 만점)
- 한 줄 평 입력 (선택)

### 필터 시트 (모바일)
- 바텀시트, max-h-85vh
- 장르/시간/인원/가격 칩 선택
- 초기화 + 적용하기 버튼

## 6. 애니메이션

```
transition-duration: 150ms (빠름) / 200ms (기본) / 300ms (느림)
easing: cubic-bezier(0.4, 0, 0.2, 1)  // ease-in-out

페이지 전환: fade + slide (100ms)
모달 열기: scale(0.95→1) + fade (150ms)
바텀시트: slide-up (250ms)
카드 호버: translate + shadow (150ms)
스켈레톤: shimmer animation (1.5s loop)
```

## 7. 아이콘

- **라이브러리**: Lucide React (일관성)
- **크기**: 14px (tiny) / 16px (body) / 18px (h3) / 20px (h2) / 24px (h1)
- **컬렉션 상태 아이콘**: 커스텀 이모지 또는 Lucide 조합
