# BoardLog — 플랫폼 비전 & 개요

> boardlife.co.kr를 레퍼런스로 삼아, 더 현대적인 UX와 트렌디한 디자인으로 보드게임 통합 플랫폼을 구축한다.

---

## 1. 서비스 정의

**BoardLog**는 보드게임 커뮤니티 + 기록 + 탐색 + 쇼핑 + 로컬을 하나로 묶은 **보드게임 종합 플랫폼**이다.

### 핵심 가치
| 가치 | 설명 |
|------|------|
| 🎲 **발견** | 방대한 게임 DB, 랭킹, 큐레이션으로 새로운 게임 발견 |
| 📝 **기록** | 플레이 로그, 평점, 컬렉션으로 내 게임 히스토리 관리 |
| 🤝 **커뮤니티** | 후기, 정보, 모임, 중고거래로 보드게이머 연결 |
| 🏪 **쇼핑** | 정가 + 할인 + 중고 + 가격비교로 최저가 구매 |
| 📍 **로컬** | 근처 보드게임 카페, 모임, 매장 탐색 |

---

## 2. 레퍼런스 분석 (boardlife.co.kr)

### 잘 된 것 (계승)
- 게임별 컬렉션 상태 8종 (보유/팬/구독/위시/선주문/펀딩/방출예정/방출)
- 커뮤니티 실시간 PLAY 랭킹
- 로컬 지도 기반 카페/모임 탐색
- 중고거래 가격비교 + 직거래
- 셀렉션 (사용자 정의 게임 그룹)

### 개선할 것 (차별화)
- **디자인**: 2010년대 구형 UI → 2025년 트렌디한 모던 UI
- **모바일 UX**: PC 중심 → Mobile-first 반응형
- **온보딩**: 복잡한 진입 장벽 → 비로그인 탐색 우선
- **검색**: 단순 텍스트 → 고도화 필터 + 추천
- **소셜**: 단방향 정보 → 팔로우 피드 + 인터랙션

## 3. 전체 서비스 맵

```
BoardLog
├── 홈 (/)                        # 큐레이션 피드, 실시간 랭킹, 배너
│
├── 보드게임 (/games)
│   ├── 탐색 (/games/explore)     # 전체 게임 DB 브라우징 + 필터
│   ├── 랭킹 (/games/ranking)     # 주간/월간 인기순위
│   ├── 신작 (/games/new)         # 최신 출시 게임
│   └── 상세 (/games/[id])        # 게임 정보 + 평점 + 커뮤니티
│
├── 커뮤니티 (/community)
│   ├── 피드 (/community)         # 최신/베스트/팔로우 글
│   ├── 후기 (/community/review)
│   ├── 정보 (/community/info)
│   ├── 공략 (/community/guide)
│   ├── 자유 (/community/free)
│   └── 글 상세 (/community/[id])
│
├── 모임 (/meetup)
│   ├── 전체 (/meetup)
│   ├── 모임 찾기 (/meetup/find)
│   └── 모임 만들기 (/meetup/create)
│
├── 로컬 (/local)
│   ├── 전체 지도 (/local)
│   ├── 카페 (/local/cafe)
│   ├── 매장 (/local/store)
│   └── 상세 (/local/[id])
│
├── 쇼핑 (/shop)
│   ├── 전체 (/shop)
│   ├── 신규 (/shop/new)
│   ├── 인기 (/shop/popular)
│   ├── 할인 (/shop/sale)
│   └── 상품 상세 (/shop/[id])
│
├── 중고 (/used)
│   ├── 전체 (/used)
│   ├── 가격비교 (/used/price)
│   ├── 직거래 (/used/direct)
│   └── 글 상세 (/used/[id])
│
└── 내 게임 (/my)
    ├── 컬렉션 (/my/collection)
    ├── 플레이 기록 (/my/plays)
    ├── 평점 (/my/ratings)
    ├── 셀렉션 (/my/selections)
    ├── 위시리스트 (/my/wishlist)
    └── 구매내역 (/my/purchases)
```

## 4. 핵심 사용자 유형

| 유형 | 특징 | 주요 사용 기능 |
|------|------|--------------|
| 🔍 **탐색형** | 새 게임 발견에 관심 | 랭킹, 탐색, 게임 상세 |
| 📚 **기록형** | 플레이 이력 관리 | 컬렉션, 플레이 기록, 평점 |
| 💬 **커뮤니티형** | 정보 공유/소통 | 게시글, 후기, 모임 |
| 🛒 **구매형** | 게임 구매/거래 | 쇼핑, 중고거래, 가격비교 |
| 📍 **로컬형** | 오프라인 활동 | 카페 탐색, 모임 참여 |

## 5. 기술 스택

| 영역 | 기술 |
|------|------|
| 프레임워크 | Next.js 16 (App Router) |
| UI | React 19 + Tailwind v4 |
| 컴포넌트 | shadcn/ui (base-ui 기반) |
| 상태관리 | Zustand + TanStack Query |
| 인증 | NextAuth.js (Google, Kakao, 이메일) |
| DB | PostgreSQL + Prisma |
| 이미지 | Cloudflare Images |
| 지도 | Kakao Maps API |
| 검색 | Typesense (자체) / BGG API 연동 |
| 배포 | Vercel + PlanetScale |
