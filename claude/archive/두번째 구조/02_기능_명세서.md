# 기능 명세서 — 섹션별 상세

---

## 1. 홈 (/)

### 섹션 구성
1. **히어로 배너 슬라이드** — 신작/이벤트/기획 배너 (자동 재생)
2. **실시간 인기 랭킹** — 이번 주 TOP 10 (순위 변동 표시)
3. **최근 평가된 게임** — 커뮤니티 최신 평가
4. **추천 컬렉션** — 에디터 큐레이션 (테마별)
5. **최근 커뮤니티 글** — 베스트 게시글 미리보기
6. **내 게임 피드** (로그인 시) — 팔로우 유저 활동, 내 최근 기록
7. **로컬 추천** — 내 위치 기반 근처 카페

## 2. 보드게임 (/games)

### 게임 탐색 필터
| 필터 | 옵션 |
|------|------|
| 인원수 | 1인, 2인, 3-4인, 5인+ |
| 플레이시간 | ~30분, 30~60분, 60~120분, 2시간+ |
| 가격대 | ~2만원, 2~4만원, 4~6만원, 6만원+ |
| 장르 | 전략, 가족, 파티, 추리, 협력, 경제, 카드, 워게임, 추상, 테마 |
| 연령 | 전체, 8+, 12+, 14+, 18+ |
| 정렬 | 인기순, 평점순, 신규순, 플레이순 |

### 게임 카드 (리스트/그리드)
- 커버 이미지 (3:4 비율)
- 게임명 (한글 + 영문)
- 평점 (별점 + 숫자)
- 인원수 / 플레이시간
- 가격
- 컬렉션 상태 뱃지 (보유/팬/위시 등)
- 빠른 컬렉션 추가 버튼 (+)

### 게임 상세 (/games/[id])
#### 탭 구조
1. **정보 탭** — 규칙 요약, 디자이너, 출판사, 수상 이력, 사진
2. **평가 탭** — 커뮤니티 평점 분포, 유저 평가 목록
3. **플레이 탭** — 커뮤니티 플레이 기록 피드
4. **구매 탭** — 정가 쇼핑 링크, 중고 시세, 가격 비교
5. **커뮤니티 탭** — 이 게임 관련 게시글

#### 액션 버튼 (로그인 시)
- 컬렉션에 추가 (상태 선택: 보유/팬/위시/...)
- 별점 평가
- 플레이 기록 추가
- 위시리스트 추가
- 공유하기

### 랭킹 (/games/ranking)
- 탭: 주간 / 월간 / 전체
- 게임별 이전 대비 순위 변동 (+n / -n / NEW)
- 인원수/장르 필터

## 3. 커뮤니티 (/community)

### 게시판 분류
| 게시판 | 내용 |
|--------|------|
| 후기 | 게임 플레이 후기 (게임 태그 필수) |
| 정보 | 신작 정보, 업데이트 소식 |
| 공략/팁 | 전략, 규칙 설명, 꿀팁 |
| 자유 | 자유 주제 |

### 피드 탭
- **최신**: 시간순 정렬
- **베스트**: 추천수 기준
- **팔로우**: 팔로우한 유저 글만

### 게시글 카드
- 게시판 태그 + 게임 태그 (있는 경우)
- 썸네일 이미지 (있는 경우)
- 제목 + 본문 미리보기
- 작성자 프로필 + 작성시간
- 👍 추천수 / 💬 댓글수 / 👁 조회수

### 게시글 상세
- 본문 (마크다운 렌더링)
- 첨부 이미지 갤러리
- 연관 게임 카드 링크
- 댓글 (대댓글 1depth)
- 추천/비추천 / 공유 / 신고
- 작성자 다른 글 보기

### 글 작성 에디터
- 리치 텍스트 에디터 (Bold/Italic/링크/이미지)
- 게임 태그 추가 (게임 검색)
- 게시판 선택
- 이미지 업로드 (최대 10장)

## 4. 컬렉션 상태 정의 (8종)

| 상태 | 아이콘 | 설명 |
|------|--------|------|
| 보유 | 📦 | 현재 소유 중인 게임 |
| 팬 | ❤️ | 좋아하는 게임 (찜) |
| 플레이해봄 | ✅ | 해본 적 있는 게임 |
| 위시리스트 | ⭐ | 구매 희망 |
| 선주문 | 📋 | 예약 주문 완료 |
| 펀딩 | 🚀 | 크라우드펀딩 참여 |
| 방출예정 | 📤 | 판매/나눔 예정 |
| 방출 | ✔️ | 판매/나눔 완료 |

### 셀렉션 (사용자 정의 그룹)
- 이름, 설명, 커버 이미지 설정
- 공개/비공개 설정
- 순서 드래그 조정
- 공유 URL 생성

## 5. 내 게임 (/my)

### 플레이 기록 필드
| 필드 | 타입 | 필수 |
|------|------|------|
| 게임 | Game | ✅ |
| 플레이 날짜 | Date | ✅ |
| 플레이어 수 | number | ✅ |
| 함께한 사람 | string[] | ❌ |
| 장소 | string | ❌ |
| 내 별점 | 1~5 | ❌ |
| 승자 | string | ❌ |
| 점수 기록 | JSON | ❌ |
| 메모 | text | ❌ |
| 사진 | image[] | ❌ |

### 통계 대시보드
- 총 플레이 횟수 / 총 플레이 시간
- 가장 많이 플레이한 게임 TOP 5
- 월별 플레이 히트맵 (GitHub 스타일)
- 함께한 사람 TOP 5
- 즐겨찾는 장소 TOP 3

## 6. 로컬 (/local)

### 장소 정보 구조
| 필드 | 설명 |
|------|------|
| 이름 | 장소명 |
| 유형 | 카페/매장/모임 |
| 주소 | 도로명 주소 |
| 좌표 | lat/lng |
| 운영시간 | 요일별 |
| 가격 | 입장료/시간당 요금 |
| 보유 게임 | 게임 목록 태그 |
| 사진 | 최대 20장 |
| 연락처 | 전화/SNS |
| 리뷰 | 유저 작성 |
| 영업 상태 | 영업중/영업종료 |

### 지도 뷰 / 리스트 뷰 전환
- 카카오맵 API 연동
- 현재 위치 기반 반경 필터
- 클러스터링 마커
- 지역 드롭다운 (시/도 → 구/군)

## 7. 쇼핑 (/shop)

### 상품 카드
- 게임 커버 이미지
- 게임명
- 정가 / 할인율 / 판매가
- 판매처 (예: 코리아보드게임즈, 다이브다이스)
- 배송 정보

### 필터
- 가격대 (슬라이더)
- 인원수
- 플레이시간
- 장르
- 판매처

### 부가 기능
- 위시리스트 연동 (내 위시리스트만 보기)
- 가격 변동 알림 등록
- 여러 판매처 가격 비교

## 8. 중고거래 (/used)

### 거래 유형
| 유형 | 뱃지 색상 |
|------|---------|
| 판매 | 파란색 |
| 구매 | 초록색 |
| 교환 | 주황색 |
| 나눔 | 보라색 |
| 완료 | 회색 |

### 글 작성 필드
- 거래 유형 선택
- 게임 태그 (검색으로 추가)
- 가격 (나눔이면 0)
- 상태 (미개봉/상/중/하)
- 거래 방식 (택배/직거래/둘다)
- 위치 (직거래 시)
- 사진 (최대 5장)
- 본문

### 가격비교 탭
- 게임별 최근 중고 거래가 모아보기
- 최저가 / 평균가 / 최고가 표시
- 시세 그래프 (최근 3개월)

## 9. 모임 (/meetup)

### 모임 정보 구조
| 필드 | 설명 |
|------|------|
| 모임명 | 이름 |
| 유형 | 정기모임/번개/온라인 |
| 게임 태그 | 주로 하는 게임 |
| 날짜/시간 | 일시 |
| 장소 | 주소 또는 로컬 연동 |
| 모집 인원 | 최소/최대 |
| 참가비 | 무료/유료 |
| 난이도 | 입문/중급/고급 |
| 설명 | 자유 텍스트 |

### 참가 플로우
1. 모임 상세에서 "참가 신청"
2. 주최자 수락/거절 또는 자동 수락
3. 알림으로 결과 통보
4. 참가자 목록 표시
