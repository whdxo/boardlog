# 내 게임 기능 소개

> 이 문서는 "내 게임" 기능이 어떤 화면으로 구성되어 있는지,
> 사용자 입장에서 어떻게 보이고 동작하는지 설명합니다.

---

## 1. 내 게임 메인 (`/my` — 컬렉션 탭)

로그인한 사용자가 자신의 보드게임 컬렉션을 한눈에 볼 수 있는 화면입니다.

### 화면 구성

```
┌─────────────────────────────────────────────┐
│  [프로필사진]  보드게임러버        [프로필 수정]│
│              보드게임과 함께하는 일상         │
│                                             │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐       │
│  │ 🎲   │ │ 📖   │ │ ⭐   │ │ 📋   │       │
│  │  5   │ │  12  │ │  8   │ │  2   │       │
│  │ 게임  │ │ 기록  │ │ 평점  │ │셀렉션│       │
│  └──────┘ └──────┘ └──────┘ └──────┘       │
│                                             │
│  [        📊 내 게임 통계 보기        ]      │
├─────────────────────────────────────────────┤
│  컬렉션    기록    평점    셀렉션             │
│  ━━━━━━                                     │
├─────────────────────────────────────────────┤
│  전체 5  📦보유중 3  ❤️팬 1  ⭐위시 1        │
│                                             │
│  내 전체 컬렉션                    최신순 ▾  │
│                                             │
│  ┌─────────┐  ┌─────────┐                  │
│  │ [썸네일] │  │ [썸네일] │                  │
│  │         │  │         │                  │
│  │ 카탄     │  │ 팬데믹   │                  │
│  │ Catan   │  │ Pandemic│                  │
│  │ ★4.5 3~4│  │ ★4.3 2~4│                  │
│  └─────────┘  └─────────┘                  │
│  ┌─────────┐  ┌─────────┐                  │
│  │ [썸네일] │  │ [썸네일] │                  │
│  │         │  │         │                  │
│  │ 스플렌더  │  │ 윙스팬   │                  │
│  │ Splendor│  │ Wingspan│                  │
│  │ ★4.2 2~4│  │ ★4.7 1~5│                  │
│  └─────────┘  └─────────┘                  │
└─────────────────────────────────────────────┘
```

### 설명

- **프로필 영역**: 내 닉네임, 자기소개, 프로필 사진이 표시됩니다.
- **통계 카드**: 내가 등록한 게임 수, 플레이 기록 수, 남긴 평점 수, 만든 셀렉션 수가 보입니다.
- **상태 필터 칩**: "전체", "보유중", "팬", "위시리스트" 등을 누르면 해당 상태의 게임만 필터링됩니다.
  - 보유중 — 내가 갖고 있는 게임
  - 팬 — 좋아하는 게임
  - 위시리스트 — 갖고 싶은 게임
  - 플레이완료 — 해본 게임
  - 예약중 — 구매 예약한 게임
  - 판매중 — 중고로 내놓은 게임
  - 빌려줌 / 빌림 — 빌려주거나 빌린 게임
- **게임 카드**: 각 카드에 썸네일, 한국어 이름, 영문 이름, 평점, 인원수가 표시됩니다.
- **정렬**: 최신순 / 이름순으로 바꿀 수 있습니다.
- 게임 카드를 누르면 → **게임별 기록 상세** 페이지로 이동합니다.

## 2. 게임별 기록 상세 (`/my/games/{게임ID}`)

특정 게임에 대한 내 플레이 기록과 통계를 보여주는 화면입니다.
보드라이프의 "게임 오브 게임즈" 상세 화면과 같은 역할입니다.

### 화면 구성

```
┌─────────────────────────────────────────────┐
│  ←  카탄                                    │
├─────────────────────────────────────────────┤
│                                             │
│  [게임 이미지]  카탄                         │
│                Catan                        │
│                👥 3-4명 · ⏱ 60-120분        │
│                                             │
│  첫 플레이:         2026-01-15              │
│  총 플레이 횟수:     12회                    │
│  내 평가 평점:      ★4.25                   │
│                                             │
├─────────────────────────────────────────────┤
│  최고 점수 │ 최저 점수 │ 평균 점수 │ 최다 승리│
│    95     │    32    │   68.5   │  홍길동  │
│           │          │          │   5회   │

│                                             │
│  👤 플레이어 통계                            │
│  ┌──────────────────────────────────────┐   │
│  │ 이름    │ 승리 │ 승률  │ 평점 │ 최고 │   │

│  │ 이영희  │  3   │ 25%  │ 4.0  │  82  │   │
│  └──────────────────────────────────────┘   │
│                                             │
├─────────────────────────────────────────────┤
│                                             │
│  👥 인원별 평점                              │

│  │ 인원    │ 플레이수 │ 내 평점  │           │

│  │ 📍 집                — 3회  │           │
│  │ 📍 친구집             — 2회  │           │
│  └──────────────────────────────┘           │
│                                             │
├─────────────────────────────────────────────┤
│                                             │
│  📝 플레이 로그                              │

│                                             │

│  ★5.00/5 · 3인플                       >   │
│                                             │
│  일요일  2026-03-09                          │
│  ★3.50/5 · 4인플                       >   │
│                                             │
│              [ 1 ] [ 2 ]                    │
│                                             │
└─────────────────────────────────────────────┘
```

### 설명

- **게임 정보**: 게임 이미지, 한/영 이름, 인원수, 플레이 시간이 표시됩니다.
- **플레이 요약**: 이 게임을 처음 한 날짜, 총 몇 번 했는지, 내가 매긴 평점이 보입니다.
- **점수 통계**: 내가 기록한 게임 점수 중 최고/최저/평균과, 가장 많이 이긴 사람이 나옵니다.
- **플레이어 통계**: 같이 게임한 사람들의 승리 횟수, 승률, 평균 점수가 테이블로 보입니다.
- **인원별 평점**: "3명이서 할 때 평점 3.8", "4명이서 할 때 평점 4.5" 처럼 인원수에 따른 만족도를 확인할 수 있습니다.
- **장소 통계**: 어디서 주로 플레이했는지 장소별로 보여줍니다.
- **플레이 로그**: 날짜별 기록 목록입니다. 누르면 상세 기록으로 이동합니다. 페이지 번호로 이전 기록도 볼 수 있습니다.

## 3. 공개 프로필 (`/u/{닉네임}`)

다른 사람의 컬렉션을 구경할 수 있는 공개 페이지입니다.
내 프로필 링크를 친구에게 공유하면 이 페이지가 열립니다.

### 화면 구성

```
┌─────────────────────────────────────────────┐
│                                             │
│  [프로필사진]  김보드                  [공유]  │
│              전략 게임을 좋아합니다           │
│                                             │
│  🎲 컬렉션 12개                              │
│                                             │
├─────────────────────────────────────────────┤
│  전체 12  📦보유중 8  ❤️팬 2  ⭐위시 2       │
├─────────────────────────────────────────────┤
│  내 전체 컬렉션                    최신순 ▾  │
│                                             │
│  ┌─────────┐  ┌─────────┐                  │
│  │ [썸네일] │  │ [썸네일] │                  │
│  │         │  │         │                  │
│  │ 카탄     │  │ 아줄     │                  │
│  │ Catan   │  │ Azul    │                  │
│  │ ★4.5 3~4│  │ ★4.3 2~4│                  │
│  └─────────┘  └─────────┘                  │
│        ...                                  │
└─────────────────────────────────────────────┘
```

### 설명

- 누구나 URL만 있으면 볼 수 있습니다 (로그인 필요 없음).
- 공유 버튼(🔗)을 누르면 현재 페이지 링크가 복사됩니다.
- 수정/삭제 버튼은 보이지 않습니다 (읽기 전용).
- 게임 카드를 누르면 그 사람의 게임별 기록을 볼 수 있습니다.

### 공유 URL 예시

```
https://boardlog.com/u/김보드
https://boardlog.com/u/김보드/games/abc-123
```

## 4. 데이터가 없을 때

아직 게임을 등록하지 않았거나 기록이 없으면 아래와 같이 보입니다.

```
┌─────────────────────────────────────────────┐
│                                             │
│               📦                            │
│                                             │
│         컬렉션이 비어있어요                   │
│    게임을 추가해 컬렉션을 채워보세요           │
│                                             │
│          [ 게임 탐색하기 ]                    │
│                                             │
└─────────────────────────────────────────────┘
```

각 상황별 안내 메시지:

| 상황 | 메시지 |
|------|--------|
| 컬렉션 비어있음 | "컬렉션이 비어있어요" + 게임 탐색 버튼 |
| 플레이 기록 없음 | "아직 플레이 기록이 없어요." |
| 점수 데이터 없음 | 점수 통계 영역이 표시되지 않음 |
| 장소 데이터 없음 | "장소별 통계가 없어요." |
| 다른 사람 프로필 없음 | "사용자를 찾을 수 없어요" + 홈 버튼 |

## 5. 페이지 이동 흐름

사용자가 이 기능을 어떤 순서로 사용하게 되는지 나타낸 흐름입니다.

```
홈(/)
 │
 └─ 하단 탭바 "내 게임" 클릭
     │
     ▼
  /my (내 게임 메인)
     │
     ├─ 컬렉션 탭 → 상태 필터(보유/팬/위시...) → 게임 카드 클릭
     │                                            │
     │                                            ▼
     │                                     /my/games/{id}
     │                                     (게임별 기록 상세)
     │                                            │
     │                                            └─ 플레이 로그 클릭
     │                                                    │
     │                                                    ▼
     │                                             /my/logs/{id}
     │                                             (기록 상세)
     │
     ├─ 기록 탭 → 월별 기록 목록
     ├─ 평점 탭 → 평점 목록
     └─ 셀렉션 탭 → 셀렉션 목록


다른 사람 프로필 공유:

  /u/{닉네임} (공개 프로필)
     │
     └─ 게임 카드 클릭
         │
         ▼
  /u/{닉네임}/games/{id} (공개 게임 기록)
```

## 6. 현재 상태와 앞으로 해야 할 것

### 지금 되는 것

| 기능 | 상태 |
|------|------|
| 내 컬렉션 목록 보기 (상태 필터, 정렬) | ✅ 완료 |
| 게임별 기록 상세 (통계, 로그) | ✅ 완료 |
| 다른 사람 컬렉션 보기 (공개 프로필) | ✅ 완료 |
| 다른 사람 게임 기록 보기 | ✅ 완료 |

### 아직 안 되는 것

| 기능 | 왜 안 되는지 | 해결 방법 |
|------|------------|----------|
| 게임 추가 | 게임 데이터가 비어있음 | BGG(보드게임 데이터베이스) 연동해서 게임 데이터를 가져와야 함 |
| 컬렉션에 게임 넣기/빼기 | 추가/삭제 기능 미연결 | 컬렉션 추가 화면과 DB 연결 필요 |
| 플레이 기록 작성 | 기록 작성 화면은 있지만 DB 저장 미연결 | 기록 저장 기능 연결 필요 |
| 기록/평점/셀렉션 탭 | 아직 샘플 데이터 표시 중 | DB에서 실제 데이터를 불러오도록 교체 필요 |
| 공개 프로필에서 기록/평점 보기 | 컬렉션만 공개됨 | 기록, 평점, 셀렉션 탭 추가 필요 |
| 팔로우 기능 | 다른 사람 팔로우/언팔로우 | 팔로우 버튼과 DB 연결 필요 |
| 내 프로필 공유 버튼 | 내 페이지에 공유 버튼 없음 | 공유 버튼 추가 필요 |
| 게임 설명 영상 | 영상 넣을 자리 없음 | 게임 상세 페이지에 영상 섹션 추가 필요 |

### 작업 순서 (권장)

```
1단계: 게임 데이터 먼저 넣기
       BGG에서 게임 정보를 가져와서 DB에 저장
       → 이게 되어야 컬렉션 추가, 기록 작성이 가능해짐
            │
            ▼
2단계: 게임 추가/기록 작성 연결
       컬렉션에 게임 추가, 플레이 기록 저장이 실제로 되게 하기
            │
            ▼
3단계: 나머지 탭 연결 + 공개 프로필 완성
       기록/평점/셀렉션 탭을 실 데이터로 교체
       공개 프로필에도 같은 탭 추가
       팔로우 기능
            │
            ▼
4단계: 마무리
       게임 설명 영상, 로딩 화면 개선, 무한 스크롤 등
```
