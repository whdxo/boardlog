# 권한 가드 구현 완료

> 브랜치: `feature/my-games-collection`
> 작업일: 2026-04-10

---

## 1. 무엇을 했는지

비회원(로그아웃 상태)이 로그인이 필요한 페이지나 버튼에 접근했을 때,
적절한 안내를 보여주도록 15개 페이지를 수정했다.

---

## 2. 변경 전 → 변경 후

### 작성/설정 페이지 (11개)

비회원이 주소를 직접 입력해서 접근할 때:

| 페이지 | 변경 전 | 변경 후 |
|--------|--------|--------|
| `/settings` | 설정 메뉴가 그대로 보임 | "로그인이 필요한 기능이에요" 안내 표시 |
| `/settings/account` | 계정 정보(이메일 등)가 보임 | 같은 안내 |
| `/settings/password` | 비밀번호 변경 폼이 보임 | 같은 안내 |
| `/settings/notifications` | 알림 설정 토글이 보임 | 같은 안내 |
| `/settings/withdraw` | 계정 삭제 화면이 보임 | 같은 안내 |
| `/profile/edit` | 프로필 수정 폼이 보임 | 같은 안내 |
| `/my/logs/write` | 기록 작성 폼이 보임 | "플레이 기록을 작성하려면 로그인해주세요" |
| `/my/logs/{id}/edit` | 기록 수정 폼이 보임 | "기록을 수정하려면 로그인해주세요" |
| `/my/selections` | 셀렉션 목록이 보임 | "셀렉션을 관리하려면 로그인해주세요" |
| `/community/new` | 게시글 작성 폼이 보임 | "글을 작성하려면 로그인해주세요" |
| `/used/write` | 거래글 작성 폼이 보임 | "거래글을 작성하려면 로그인해주세요" |

안내 화면에는 **로그인/회원가입 버튼**이 함께 표시된다.

### 공개 페이지의 버튼 (4개)

페이지 자체는 누구나 볼 수 있지만, 특정 버튼은 로그인이 필요한 경우:

| 페이지 | 버튼 | 변경 전 | 변경 후 |
|--------|------|--------|--------|
| `/games/{id}` | "컬렉션 추가" | 컬렉션 시트가 열림 (저장 안 됨) | 로그인 유도 시트가 올라옴 |
| `/games/{id}` | "평점" | 평점 모달이 열림 (저장 안 됨) | 로그인 유도 시트가 올라옴 |
| `/community/{id}` | 좋아요 | 아무 반응 없음 | 로그인 유도 시트가 올라옴 |
| `/community/{id}` | 댓글 작성 | 입력은 되지만 제출 안 됨 | 로그인 유도 시트가 올라옴 |
| `/used/{id}` | 댓글 작성 | 입력은 되지만 제출 안 됨 | 로그인 유도 시트가 올라옴 |
| `/my/selections/{id}` | ⋯ 메뉴 | 누구에게나 보임 | 로그인한 사람에게만 보임 |

로그인 유도 시트에서 로그인하면 **원래 보던 페이지로 자동 복귀**한다.

## 3. 새로고침/뒤로가기 처리

| 상황 | 동작 |
|------|------|
| **새로고침 (F5)** | 로그인 확인이 끝날 때까지 빈 화면 → 확인 후 정상 표시. 폼이 잠깐 보였다 사라지는 현상 없음 |
| **뒤로가기** | 로그인 상태가 메모리에 남아있어서 즉시 판단. 깜빡임 없음 |
| **로그인 후 새로고침** | 쿠키에 세션이 저장되어 있어서 자동으로 로그인 상태 복원 |
| **세션 만료 후 접근** | 로그인 확인 실패 → 안내 화면 표시 |

## 5. 전체 페이지 권한 현황 (최종)

```
✅ 로그인 가드 있음 (안전):
  ├── /my (컬렉션/기록/평점/셀렉션)
  ├── /my/games/[id] (게임별 기록)
  ├── /my/logs/write (기록 작성)
  ├── /my/logs/[id]/edit (기록 수정)
  ├── /my/selections (셀렉션 목록)
  ├── /profile (내 프로필)
  ├── /profile/edit (프로필 수정)
  ├── /settings (설정 메인)
  ├── /settings/account (계정 정보)
  ├── /settings/password (비밀번호 변경)
  ├── /settings/notifications (알림 설정)
  └── /settings/withdraw (회원 탈퇴)

✅ 버튼 레벨 가드 있음:
  ├── /games/[id] — 컬렉션/평점 버튼
  ├── /community/[id] — 좋아요/댓글
  ├── /used/[id] — 댓글
  └── /my/selections/[id] — 수정/삭제 메뉴

✅ 공개 페이지 (가드 불필요):
  ├── /games, /games/[id], /games/new, /games/ranking
  ├── /community, /community/[id]
  ├── /local, /local/[id]
  ├── /shop, /shop/[id]
  ├── /used, /used/[id]
  ├── /search
  ├── /u/[nickname], /u/[nickname]/games/[id]
  ├── /login, /signup, /forgot-password, /reset-password
  └── /terms, /privacy, /guidelines
```
