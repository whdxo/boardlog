# BGG 데이터 → BoardLog 매핑 & 활용

> BGG에서 가져온 데이터를 우리 DB와 UI에 어떻게 활용하는지 정리

---

## 1. DB 매핑 (BGG → board_games 테이블)

### 기본 매핑

| BGG 필드 | 우리 DB 필드 | 변환 | 비고 |
|----------|-------------|------|------|
| `item@id` | `bgg_id` | 그대로 | 연동 키 |
| `name[@type='primary']` | `title` | 그대로 | 영문 기본 이름 |
| `name[@type='alternate']` (한국어) | `title_en` → 사실상 `title_kr` 추가 필요 | 한국어 필터링 | 한글 이름 있으면 저장 |
| `description` | `description` | HTML 디코딩 | `&amp;` → `&` 등 |
| `image` | `image_url` | 그대로 | BGG CDN URL |
| `minplayers` | `min_players` | 정수 | |
| `maxplayers` | `max_players` | 정수 | |
| `minplaytime` | `play_time_min` | 정수 (분) | |
| `maxplaytime` | `play_time_max` | 정수 (분) | |
| `minage` | `min_age` | 정수 | |
| `yearpublished` | `year_published` | 정수 | |
| `link[@type='boardgamedesigner'][0]` | `designer` | 첫 번째 값 | |
| `link[@type='boardgamepublisher'][0]` | `publisher` | 첫 번째 값 | |
| `statistics/ratings/average` | `avg_rating` | **÷2** (10점→5점) | 소수점 1자리 |
| `statistics/ratings/usersrated` | `rating_count` | 정수 | |

### 추가 매핑 (tags 테이블)

| BGG 필드 | tags.type | 예시 |
|----------|-----------|------|
| `link[@type='boardgamecategory']` | `genre` | 전략, 가족, 파티, 추리 |
| `link[@type='boardgamemechanic']` | `mechanic` | 주사위, 카드 드래프트, 워커 플레이스먼트 |

### DB 스키마 변경 제안

현재 `board_games` 테이블에 없는 유용한 필드들:

```
board_games (추가 제안)
├── title_kr (string, nullable)        ← 한국어 이름
├── thumbnail_url (string, nullable)   ← 썸네일 (목록용 작은 이미지)
├── bgg_rank (int, nullable)           ← BGG 전체 순위
├── weight (float, nullable)           ← 난이도 (1~5)
├── owned_count (int, 캐싱)            ← "N명이 보유"
├── wishlist_count (int, 캐싱)         ← "N명이 찜"
└── last_synced_at (datetime)          ← BGG 데이터 마지막 동기화 시각
```

## 2. UI 활용 — 어디서 어떤 데이터를 쓰나

### 게임 카드 (목록, 홈)

```
┌──────────────────┐
│ [thumbnail_url]   │  ← BGG thumbnail
├──────────────────┤
│ 카탄              │  ← title_kr (없으면 title)
│ ★ 3.6  ·  3~4인  │  ← avg_rating(변환) + min~max_players
│ 35,000원          │  ← price (우리 DB, BGG에 없음)
└──────────────────┘
```

| 표시 항목 | BGG 소스 | 변환 |
|-----------|----------|------|
| 썸네일 | `thumbnail` | 그대로 |
| 게임명 | `name[alternate]` 한국어 우선 | 없으면 영문 |
| 평점 | `ratings/average` | ÷2 |
| 인원 | `minplayers`~`maxplayers` | "3~4인" 형식 |
| 가격 | (BGG에 없음) | 우리가 입력 |

### 게임 상세 페이지

| UI 섹션 | BGG 소스 | 활용 |
|---------|----------|------|
| 메인 이미지 | `image` | 고해상도 |
| 게임명 + 영문명 | `name[primary]` + `name[alternate]` | 한/영 표시 |
| 정보 칩 | `minplayers`~`maxplayers`, `playingtime`, `minage` | [3~4인] [60~90분] [10세+] |
| 게임 소개 | `description` | HTML 디코딩 후 표시 |
| 디자이너 | `link[boardgamedesigner]` | 클라우스 토이버 |
| 퍼블리셔 | `link[boardgamepublisher]` | 코스모스 |
| 출시년도 | `yearpublished` | 1995 |
| 장르 태그 | `link[boardgamecategory]` | [전략] [협상] [경제] |
| 메카닉 태그 | `link[boardgamemechanic]` | [주사위] [교역] |
| 평점 | `ratings/average` | ★★★★☆ 3.6 (10점 기준 7.15) |
| 평가 수 | `ratings/usersrated` | "98,234명 평가" |
| 난이도 | `ratings/averageweight` | "난이도 2.3/5" |
| BGG 순위 | `ranks/rank[boardgame]` | "BGG #423" |
| 보유 수 | `ratings/owned` | "89,012명 보유" |
| 위시 수 | `ratings/wishing` | "5,678명 찜" |
| 연관/확장 | `link[boardgameexpansion]` | 확장팩 목록 |

### 추천 인원 뱃지 (Phase 2)

BGG `suggested_numplayers` 투표 데이터 활용:

```
[👥 4인 추천]  ← Best 투표가 가장 높은 인원수
[👥 3~5인 적합]  ← Recommended 이상인 인원 범위
```

### 난이도 표시 (Phase 2)

BGG `averageweight` (1~5) 활용:

```
난이도: ■■□□□ 가벼움 (1~2)
난이도: ■■■□□ 보통 (2~3)
난이도: ■■■■□ 무거움 (3~4)
난이도: ■■■■■ 매우 무거움 (4~5)
```

## 3. 검색 흐름에서의 활용

```
유저 검색 "카탄"
      │
      ▼
  우리 DB 검색 (title, title_kr LIKE '%카탄%')
      │
      ├── 결과 있음 → 바로 표시
      │
      └── 결과 부족 → BGG /search?query=catan
                         │
                         ▼
                    BGG 결과에서 ID 추출
                         │
                         ▼
                    /thing?id=13,234,567 (배치)
                         │
                         ▼
                    우리 DB에 저장 + 검색 결과에 추가
```

## 4. 홈 "인기 게임"에서의 활용

```
      │

      │
      ▼

      │
      ▼
  홈에서 표시
```

## 5. 가격 정보 (BGG에 없는 데이터)

BGG에는 가격 정보가 없음. 우리가 별도로 관리해야 하는 데이터:

| 데이터 | 소스 | 관리 방법 |
|--------|------|----------|
| 판매 가격 | 네이버 쇼핑, 쿠팡 등 | MVP: 수동 입력 / Phase 2: 크롤링 |
| 구매 링크 | 외부 쇼핑몰 URL | `purchase_links` 테이블 |
| 한국어 이름 | BGG alternate + 수동 | `title_kr` 필드 |
| 한국 퍼블리셔 | 직접 입력 | 코리아보드게임즈, 다이브다이스 등 |

## 6. 평점 변환 규칙

| BGG (10점) | BoardLog (5점) | UI 표시 |
|------------|---------------|---------|
| 8.0~10.0 | 4.0~5.0 | ★★★★☆ ~ ★★★★★ |
| 6.0~7.9 | 3.0~3.9 | ★★★☆☆ ~ ★★★★☆ |
| 4.0~5.9 | 2.0~2.9 | ★★☆☆☆ ~ ★★★☆☆ |
| 2.0~3.9 | 1.0~1.9 | ★☆☆☆☆ ~ ★★☆☆☆ |

```typescript
function convertBggRating(bggRating: number): number {
  return Math.round((bggRating / 2) * 10) / 10; // 소수점 1자리
}

// BGG 7.15 → 3.6
// BGG 8.40 → 4.2
```

> 자체 유저 평점이 10개 이상 쌓이면 자체 평점 우선 표시, BGG 평점은 "BGG 평점" 라벨로 보조 표시.

## 7. 수동 관리 데이터 전체 목록

BGG API에서 가져올 수 없어서 **우리가 직접 만들고 관리해야 하는 데이터** 전체 정리.

### 7.1 게임 관련 수동 데이터

| 데이터 | DB 위치 | 소스 | MVP 관리 방법 | Phase 2 |
|--------|---------|------|-------------|---------|
| **한국어 게임명** | `board_games.title_kr` | BGG alternate에 없는 경우 | 관리자 수동 입력 | 커뮤니티 제보 |
| **한국어 설명** | 별도 필드 추가 필요 | BGG는 영문만 | 인기 게임 우선 번역 | AI 번역 + 수동 검수 |
| **판매 가격** | `board_games.price` | 네이버 쇼핑, 쿠팡 검색 | 수동 입력 | 가격 크롤링 자동화 |
| **구매 링크** | `purchase_links` 테이블 | 쇼핑몰 URL | 수동 입력 | 자동 수집 |
| **구매처 이름** | `purchase_links.shop_name` | — | 수동 (네이버, 쿠팡 등) | — |
| **재고/품절 상태** | `purchase_links.is_available` | — | 수동 확인 | 자동 체크 |
| **국내 전용 게임 전체** | `board_games` 전체 필드 | BGG에 없는 게임 | 관리자 직접 등록 | — |
| **게임 자체 이미지** | Supabase Storage | 국내 게임 or BGG 이미지 부족 시 | 직접 촬영/제작 업로드 | — |
| **한국 퍼블리셔 정보** | `board_games.publisher` | — | 수동 (코보게, 다이브다이스 등) | — |

### 7.3 유저 생성 데이터 (자동이지만 우리 DB에만 존재)

| 데이터 | DB 위치 | 설명 |

### 7.4 수동 데이터 우선순위 (MVP 런칭 전 필수)

```
🔴 필수 (런칭 전):
├── 인기 게임 100개 한국어 이름 입력
├── 인기 게임 100개 가격 + 구매 링크 입력
├── 이용약관 + 개인정보처리방침 작성
├── 홈 배너 최소 3장 제작
└── 추천 게임 최소 8개 선정

🟡 권장 (런칭 1주 내):
├── 인기 게임 300개 한국어 이름
├── 인기 게임 300개 가격
├── 서비스 가이드라인 작성
└── 국내 전용 인기 게임 20개 등록

🟢 점진적 (운영하면서):
├── 한국어 이름 지속 확대
├── 가격 정보 지속 확대
├── 국내 게임 지속 등록
└── 배너/추천 주기적 갱신
```
