## 3. 주요 결정 사항

- **인원수 필터 비활성화**: `min/max_players`가 전부 DB default(1, 4)로 실제 데이터 없음 → Tier 2 완료 후 활성화 예정
- **필터 버튼 항상 고정**: 장르 칩이 많아 모바일에서 스크롤 시 필터 버튼이 숨겨지는 UX 문제 해결 — 필터 버튼을 스크롤 영역 밖 우측에 고정
- **FilterSheet 재편**: 장르 필터는 FilterChips로 이동, 상세 필터(인원/가격/시간/연령)만 Sheet에서 관리
